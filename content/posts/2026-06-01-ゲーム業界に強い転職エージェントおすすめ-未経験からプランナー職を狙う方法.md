## ① ゲーム業界の転職にエージェントが必要な理由

### 一般エージェントとゲーム特化エージェントの違い

「リクルートエージェントやdodaに登録したけど、ゲーム関連の求人がほとんど出てこなかった」という経験はありませんか。これは珍しいことではなく、多くの未経験者が最初にぶつかる壁です。

一般エージェントとゲーム特化エージェントでは、そもそも保有している求人のラインナップが根本的に異なります。

| 比較項目 | 一般エージェント（例：doda） | ゲーム特化エージェント（例：G-JOB） |
|---|---|---|
| ゲーム関連求人数 | 全体の数%程度 | ほぼ全件がゲーム関連 |
| 担当者の業界知識 | 浅い（業界全般担当） | 深い（ゲーム業界特化） |
| 非公開求人へのアクセス | 少ない | 多い（大手各社からの非公開案件） |
| 未経験向けサポート | 標準的 | ポートフォリオ添削など専門サポートあり |
| 面接対策の精度 | 汎用的 | ゲーム会社の面接官傾向を熟知 |

国内のゲーム産業は市場規模2兆4,000億円（2024年、前年比3.4%増・ファミ通調べ）と成長が続いていますが、その就業人口は約20万人（CESAゲーム産業レポート2024）と決して多くない専門業界です。

「大手エージェントでもゲーム求人は出る」のは事実です。ただ担当者の業界知識の差が選考結果に直結することが多く、特に未経験の方には痛手になりやすいです。担当者の業界知識の差、思ったより影響が大きいんですよね。

{{< point >}}ゲーム業界への転職は、最初の1社目にゲーム特化エージェントを選ぶことで成功確率が大きく変わります。{{< /point >}}

### 非公開求人と業界特有の選考フローへの対応

ゲーム業界では、大手・中堅ゲーム会社の求人のかなりの割合が非公開で流通しています。応募が殺到するのを避けるためや、候補者のレベルをある程度担保したいという会社側の事情があります。

「どうせ有名なところは求人なんて出てないでしょ…」と思っている方、実はそれが誤解です。非公開求人は、エージェントに登録しないと目にすることすらできません。

また、ゲーム業界の選考には独自のフローが存在します。書類選考の段階から「ポートフォリオ・企画書の提出」を求める企業が多く、これが一般的な転職活動との最大の違いです。

1. 書類選考（職務経歴書＋ポートフォリオ審査）
2. 筆記・適性検査（論理的思考・センス確認）
3. 一次面接（部門マネージャー・リードプランナー）
4. 二次面接（役員・プロデューサー）
5. 内定

ここで準備のないまま応募すると、いくら意欲があっても書類で落とされてしまいます。

{{< box color="yellow" title="未経験者が注意すべき選考の落とし穴" >}}
- ポートフォリオなしで応募 → 書類選考で即落ち
- 企画書の品質が低い → 「本当にゲームが好きなのか？」と疑われる
- 自己PRが他業界と同じ内容 → ゲームへの熱意が伝わらない
- 応募先のジャンルを理解していない企画書 → ミスマッチと判断される
{{< /box >}}

ゲーム特化エージェントは「ポートフォリオの添削」「企業ごとの選考傾向の提供」といった一般エージェントにはできないサポートをしてくれます。これは非常に大きな差です。

{{< point >}}非公開求人へのアクセスとゲーム業界特有の選考対策が、ゲーム特化エージェントを選ぶ最大の理由です。{{< /point >}}

### ゲームプランナー職に求められるスキルとポートフォリオ

「ポートフォリオって何を作ればいいの？」これは未経験者が最も多く抱える疑問の一つです。正直なところ、「なんとなくゲームが好き」だけでは書類選考を通れません。

採用担当者が見るのは、次のようなポイントです。

- 論理的な根拠（数字・データで根拠を説明できるか）
- 応募先のジャンルに近い企画内容か
- マネタイズ視点（課金率・ユーザー獲得のKPI意識）があるか
- プロフィール・自己PR・将来ビジョンが明確か

{{< box color="green" title="ゲームプランナーのポートフォリオ必須コンテンツ" >}}
1. オリジナルゲームの企画書（ジャンル・ターゲット・マネタイズ設計）
2. 既存ゲームの改善案（課題→施策→期待効果の流れで記述）
3. 新機能提案書（ユーザー目線＋数値根拠を添える）
4. 自己PR・志望動機（なぜゲーム業界なのか、具体的に）
{{< /box >}}

「資格より企画力・コミュニケーション力が重視される」のがゲームプランナー職の特徴です。簡単なものでもよいので、1つは必ず用意してください。企画書1枚でも、作っている人と作っていない人では書類通過率が大きく変わります。

### 未経験採用している会社の傾向

競合の転職記事のほとんどは「おすすめエージェント一覧」で終わっています。この記事では一歩踏み込んで、「どんな企業が未経験プランナーを採用しているのか」の傾向を解説します。

スマホゲーム（モバイル）企業は、開発サイクルが早く人材需要が高い傾向にあります。コンシューマーゲームと比べて採用基準が柔軟なケースが多く、「ポテンシャル採用」が行われます。

ベンチャー・中小ゲーム会社は、大手に比べて採用基準が現実的です。即戦力よりも「育てる前提」の採用をする会社もあり、成長意欲や吸収力を重視します。

恋愛ゲーム・育成シミュレーション系のメーカーは、コミュニケーション力や言語化力が特に重視される業種です。エンジニア的なスキルよりも「ユーザー目線で伝える力」が求められるため、未経験でも挑戦しやすいです。

また、最初から正社員でなくてもよい方であれば、アルバイト・派遣からゲーム業界に入る方法もあります。20代未経験はプランナーアシスタントとして採用されるケースが多く、そこから正社員に登用されるルートも存在します。

{{< box color="gray" title="プランナーアシスタントから正社員へのルート" >}}
1. アルバイト・派遣でゲーム会社に入社
2. プランナーアシスタントとして現場経験を積む
3. 実績を評価され正社員へ登用

シリコンスタジオエージェントは、このような段階的なキャリア構築を専門に支援しています。
{{< /box >}}

ちなみにCESAの調査によれば、転職した開発者のうち「年収が上がった」と回答した人が77.9%（「20%以上上がった」19.5%含む）というデータがあります（CESA「ゲーム開発者の就業とキャリア形成2024」）。転職ハードルを感じている方でも、転職によって年収アップを実現している人が多いのは心強いですね。

{{< point >}}未経験プランナーを目指すなら、最初のターゲットはスマホゲーム系の中小・ベンチャーに絞るのが現実的な戦略です。{{< /point >}}

### 登録〜内定までのステップ詳細

**ステップ1：エージェントに無料登録（5〜10分）**

フォームに基本情報を入力するだけで完了します。この時点でポートフォリオがなくても問題ありません。面談の中で「これからどう準備するか」を相談できます。

**ステップ2：担当者との面談（30〜60分）**

ここが転職活動の成否を左右するキーポイントです。「現職でゲームに関わった経験はあるか」「どんなジャンルのゲームを作りたいか」「いつまでに転職したいか」などを聞かれます。

初回面談では次のことを正直に伝えてください。

- ゲーム業界での経験の有無（未経験なら素直に）
- 希望する職種（プランナー希望なら明確に）
- 年収・勤務地などの条件
- ポートフォリオの進捗状況

{{< box color="cyan" title="担当者との面談で意識したいこと" >}}
「正直に話すことが最も良い求人を引き出す」が鉄則です。未経験であることを隠すより、素直に伝えた方が担当者も適切な求人を絞り込みやすくなります。担当者が合わないと感じたら「担当変更」を申し出ることも可能です。
{{< /box >}}

**ステップ3：求人紹介と応募**

担当者があなたの希望にマッチした求人を複数提案してくれます。このとき非公開求人も含まれるため、求人サイトでは見つけられなかった案件に出会える可能性があります。「こんな企業があったの？」という発見が多いのもエージェント活用の醍醐味です。

**ステップ4：書類選考〜面接（2〜3回）**

ゲーム会社の選考は2〜3回面接が一般的です。エージェントが「この企業の面接官はどんな質問をするか」を事前に教えてくれるため、準備の精度が上がります。

**ステップ5：内定・年収交渉・入社**

内定後の年収交渉もエージェントが代行してくれます。「年収交渉ってどう切り出せば？」と悩む方は多いですが、ここは遠慮なく担当者に任せましょう。エージェント経由の方が直接応募より高い提示になりやすい傾向があります。

ちなみにゲーム業界の平均年収は708万円（CESA2024）ですが、初任給平均は23.6万円と低めです。転職の場合は前職年収を基準に交渉することが多く、プロに任せた方が安心です。

### 面談で伝えるべきこと・絶対にやってはいけないNG例

「エージェントの面談で失敗した」という声は意外と多いです。正直なところ、面談でNGな伝え方をすると担当者の優先度が下がって、良い求人が回ってこなくなることもあります。

{{< box color="red" title="面談でやってはいけないNG例" >}}
- ❌「とにかく有名なゲーム会社に行きたい」だけを強調する（現実離れと思われる）
- ❌ ポートフォリオがないのに「今すぐ転職したい」と言う（準備不足が見透かされる）
- ❌「どんな仕事でもいいです」と言う（意欲なしと判断される）
- ❌ 年収条件を最初から高く設定しすぎる（未経験では求人が絞られる）
- ❌ ゲームタイトルの名前だけを並べて志望動機にする（表面的すぎる）
{{< /box >}}

実際に聞いた体験談をひとつ紹介します。Aさん（26歳・IT業界未経験）は、初回面談で「Nintendo Switchの有名タイトルのプランナーになりたい」と伝えました。担当者には「まずは現実的なラインから始めましょう」と言われ、最初は戸惑ったそうです。

{{< point >}}面談では正直に話し、現実的な目標を設定することが、良い求人を引き出すカギです。{{< /point >}}

### Q2. 地方在住でもゲーム会社に転職できる？

できます。ただし、注意点があります。

ゲーム会社の多くは東京・神奈川・大阪に集中しており、地方での勤務は難しいケースが多いです。一方でコロナ禍以降にリモートワーク対応を進めた企業も増えており、エージェントに「フルリモート可・地方在住可」で求人を絞り込んでもらうことができます。

「地方に住んでいるけどゲームプランナーになりたい」という方には2つの選択肢があります。

1. 転職後に東京等へ移住を検討する（引越し支援制度がある企業もある）
2. フルリモート可の企業を中心に探す（求人数は少ないが存在する）

エージェントに「地方在住であること」を最初から正直に伝えることで、現実的な提案をもらえます。

### Q3. ゲーム業界は激務って本当？

分かります。「好きなことを仕事にする代償では？」と気になりますよね。正直に言えば、激務な職場も多いです。業界経験者の体験談として「マスターアップ前の残業爆増は業界あるある」「長時間労働・納期プレッシャーへの後悔」という声は実際にあります。

ただし企業によってワークライフバランスへの取り組みは大きく異なります。働き方改革を進めている会社や、モバイルゲーム系で比較的安定したペースの職場も増えています。エージェントに「残業時間・裁量労働制」について企業に確認してもらうことも可能です。

{{< box color="yellow" title="入社前に確認したいこと（ゲーム会社）" >}}
- 月の平均残業時間（マスターアップ時期を除いた平常時）
- 裁量労働制かどうか
- リモートワーク・フレックスの有無
- 過去3年以内の離職率・定着率
{{< /box >}}

「やりたいゲームの仕事に携われない」という悩みも多いです。有名タイトルに関わるには、まず業界実績を積むことが大切です。業界の外からいきなり夢のタイトルを目指すより、まず業界に足を踏み入れることが近道です。もちろん最終的にどの道を選ぶかはあなた次第ですが。

### Q4. ゲームプランナー職の将来性は？

国内ゲーム市場は2兆4,000億円（2024年）と成長が続いており、グローバルでは31兆42億円という規模に達しています（CESA・ファミ通調べ）。業界全体の成長が続いている点は心強い材料です。

一方で、2024年のグローバルレイオフは14,600人（前年比39%増）と、海外大手の余剰人員整理が続いています（games.gg調査）。これは国内大手にも影響が波及しうる点として念頭に置いておきましょう。

また、UE5などの最新エンジン活用・マルチプラットフォーム展開が加速しており、職種ごとの専門性がますます重視されています。
