「このまま働いても年収が上がる気がしない…」と感じている薬剤師の方、多いのではないでしょうか。

調剤薬局で毎日がんばっているのに、昇給は微々たるもの。都市部の高い家賃を払うと、手取りはびっくりするほど少ない。転職に興味はあるけど、情報が少なくて一歩踏み出せない。そんな状況が続いていませんか。

この記事では、転職情報を7年以上調査してきた筆者ittiが、薬剤師の年収アップに効く「在宅医療×地方転職」という掛け合わせ戦略をわかりやすく解説します。

この記事を読めば、こんな悩みが解決します。

- 「なぜ自分の年収が上がらないのか」構造的な理由がわかる
- 「在宅医療薬剤師」への転職でいくら稼げるのかリアルな数字がわかる
- 地方転職で年収150万円以上アップした実例と、失敗を防ぐ方法がわかる

丁寧に解説していくので、ぜひ最後まで読んでみてください。

## ①薬剤師の平均年収と「現状の壁」を正確に知る

### 調剤薬局 vs 病院 vs ドラッグストアの年収比較

まず現状を正確に把握するところから始めましょう。

厚生労働省「令和6年賃金構造基本統計調査」（2025年発表）によると、薬剤師の全国平均年収は 599.3万円（前年比+21.4万円）。一見、悪くない数字に見えます。

ただし職場によって大きな開きがあります。

| 職場 | 平均年収 |
|---|---|
| 調剤薬局 | 583.8万円 |
| 調剤併設型ドラッグストア | 547.0万円 |
| 病院薬剤師 | 521.7万円 |

調剤薬局と病院薬剤師の差は約62万円。「病院勤務だからこそ」と頑張っているのに、同期の調剤薬局組より60万円以上低いのは、正直しんどいですよね。

年齢別に見ると、25〜29歳で501万円、35〜39歳で614.1万円、55〜59歳で709.3万円と右肩上がりです。ただし、35歳以降の伸びは鈍くなる傾向があります。

{{< box color="blue" title="2024年度診療報酬改定のポイント" >}}
- 40歳未満の薬局薬剤師：+0.28%の処遇改善
- 病院薬剤師：+0.61%の処遇改善
- 若手薬剤師の給与改善が制度として盛り込まれた
{{< /box >}}

処遇改善は実施されましたが、0.28〜0.61%というのが現実。制度の恩恵はありますが、これだけでは劇的な年収アップは難しいというのが本音です。

{{< point >}}職場と年齢で年収が大きく変わる。今いる職場の「構造的な上限」を理解することが出発点。{{< /point >}}

### なぜ年収が上がりにくいのか？構造的な理由

「なんで昇給しないんだろう…」と感じている方に、正直に説明します。

薬剤師の年収が上がりにくい構造的な理由は主に3つあります。

1. 薬剤師数が過去最多水準（2024年：32万9,045人）で、供給が安定しているため求人側が強い
2. 調剤報酬が規制されているため、薬局の収益構造が硬直的で昇給原資が少ない
3. 専門性の差別化が難しいと認識されており、資格なしでも一定の業務ができる

つまり、同じ職場でただ年数を重ねるだけでは、賃金が伸びにくい構造になっています。これは薬剤師個人の問題ではなく、業界全体の問題です。

気持ちはよくわかります。一生懸命働いているのに報われないと感じるのは当然です。

だからこそ、{{< mark >}}環境を変える{{< /mark >}}という選択が有効になります。

## ②在宅医療薬剤師という選択肢を真剣に考える

### 在宅医療での薬剤師の役割と需要

「在宅医療薬剤師って、どんなことをするの？」という方に、簡単に説明します。

訪問薬剤師（在宅薬剤師）とは、患者さんの自宅や施設に出向き、薬の管理・服薬指導・副作用チェックをおこなう薬剤師のことです。ようするに、「薬局に来られない患者さんのもとへ、薬剤師が出向く」仕事です。

需要は急拡大しています。厚生労働省の統計によると、在宅医療対応薬局の割合は全体の43.6%（2024年8月時点）。裏を返せば、約56%の薬局はまだ在宅に対応できていません。

また、「2025年問題」として知られる団塊の世代が75歳以上になる流れが続いており、在宅訪問回数はすでに年間約900万回に達しています（推計）。この数字はこれからも増え続けます。

{{< box color="green" title="在宅医療薬剤師の需要が高い理由" >}}
- 在宅対応薬局は全体の43.6%にとどまり、残り56%に拡大余地がある
- 高齢化による在宅医療需要の増加が継続中
- 薬剤師の訪問件数は年間約900万回（推計）と増加傾向
- 在宅経験者は転職市場での評価が高く、引く手あまたの状況
{{< /box >}}

需要がある分野で経験を積むことで、市場価値が上がります。これが年収アップにつながる仕組みです。

{{< point >}}在宅医療分野は需要急拡大中。今のうちに経験を積めば、長期的な市場価値が高まる。{{< /point >}}

### 在宅専門薬局・訪問薬剤師の年収相場

「在宅に転職すると年収はどうなるの？」これが最も気になるところですよね。

在宅ファーマの調査によると、訪問薬剤師の年収相場は400万円〜600万円。都市部では600万円以上も可能で、経験5年以上であれば500万円以上が目安です。

注意点があります。調剤薬局の平均（583.8万円）と比べると、在宅薬剤師の相場は必ずしも高くありません。訪問薬剤師として転職するだけで年収が劇的に上がるわけではない、というのが正直なところです。

「じゃあ意味ないじゃないか」と思うかもしれませんが、在宅医療のメリットは年収の上昇余地にあります。

在宅経験を積んだ薬剤師は転職市場での評価が高く、次のステップ（管理薬剤師・在宅専門薬局の立ち上げ等）でより高い年収を狙えます。地方の在宅専門薬局では都市部より人材不足が深刻なため、好条件で迎えられるケースも多いです。

{{< point >}}在宅転職は即効薬より長期投資。地方の在宅専門薬局との掛け合わせが最もリターンが大きい。{{< /point >}}

### 在宅医療で求められるスキルと資格

「在宅医療に転職するために、何か特別な資格が必要？」よく聞かれる疑問です。

結論から言うと、{{< mark >}}特別な資格がなくても始められます{{< /mark >}}。ただし、以下のスキルや資格があると評価されやすくなります。

| スキル・資格 | 評価ポイント |
|---|---|
| 在宅療養支援薬局認定 | 薬局単位の認定。在宅対応の実績証明 |
| 認定薬剤師（各種） | 専門性の証明。採用時の評価UP |
| コミュニケーション能力 | 患者・家族・医師・ケアマネとの連携 |
| 医療保険・介護保険の知識 | 算定・請求業務に必須 |
| 運転免許（MT/AT） | 訪問には車移動が基本 |

資格より大切なのは、「患者さんの生活に寄り添いたい」という気持ちです。在宅薬剤師の声を聞くと、「患者さんの生活環境を把握でき、深くかかわれる」「患者・家族と信頼関係を築けることが嬉しい」という声が多く上がっています。

{{< point >}}特別資格より「人に寄り添う姿勢」が求められる。資格は在宅経験を積んでから取っても遅くない。{{< /point >}}

## ③地方転職で年収アップが狙える理由【驚きのデータあり】

### 地方特有の求人ニーズと給与水準

「地方に転職したら年収が下がる」と思っていませんか？実はそれは大きな誤解かもしれません。

厚生労働省「令和6年賃金構造基本統計調査」のデータを見ると、驚くべき事実があります。

| 都道府県 | 平均年収 | 全国平均との差 |
|---|---|---|
| 熊本県 | 761.8万円 | +162万円 |
| 広島県 | 715.7万円 | +116万円 |
| 東京都 | 589万円 | −10万円 |
| 岡山県 | 513.6万円 | −86万円 |
| 宮崎県 | 511.0万円 | −88万円 |

熊本県の薬剤師平均年収は761.8万円。東京都（589万円）より172万円も高い水準です。

「なぜ地方で高い？」という疑問はもっともです。理由は薬剤師不足。都市部は薬剤師が集まりやすいため競争が激しく、賃金が下がりやすいです。一方、地方では薬剤師の確保が難しいため、採用のために給与を上げざるを得ない状況があります。

ただし、地方によって差が大きいことも事実です。熊本・広島のような高年収エリアがある一方、岡山・宮崎のように全国平均を下回るエリアもあります。「地方なら必ず高い」とは言えません。転職先の地域リサーチは必須です。

{{< box color="yellow" title="地方転職で年収が高いエリアを狙うコツ" >}}
- 九州・中国地方は薬剤師不足が深刻で高年収求人が多い傾向
- 政令指定都市（熊本市・広島市等）は都市機能を保ちながら高年収が狙える
- 農村部は年収が高くても生活インフラが整っていない場合があるため事前確認が必須
{{< /box >}}

{{< point >}}地方転職の成功の鍵は「薬剤師不足エリア」の選定。熊本・広島などは全国平均を大きく超える。{{< /point >}}

### 生活コスト削減で実質年収が上がる仕組み

ここが他の記事があまり触れていない、この記事の独自の切り口です。

年収は「名目年収」だけで比較してはいけません。大切なのは「手取りベース」、つまり生活費を差し引いた実質的な豊かさです。

たとえば、東京都心のワンルームマンションの相場は月10〜15万円。一方、地方都市（熊本市・広島市など）では月5〜8万円程度です。年間で見ると60〜84万円の差になります。

東京で年収589万円 vs 熊本で年収762万円を比較してみましょう。

| 項目 | 東京 | 熊本 | 差額 |
|---|---|---|---|
| 年収（平均） | 589万円 | 762万円 | +173万円 |
| 家賃（年間） | 144万円 | 84万円 | −60万円 |
| 食費・交通費（概算） | 120万円 | 90万円 | −30万円 |
| 実質手取り差 | − | − | 約+263万円 |

名目の年収差173万円に加え、生活コスト削減分を合わせると実質的な差は260万円超になる計算です。

もちろん人によって生活スタイルは異なります。ただ、「年収が同じでも地方のほうが豊かに暮らせる」という視点は、転職判断に取り入れる価値があります。

{{< point >}}地方転職は年収の数字だけでなく「手取りベース」で比較する。生活コスト込みで考えると地方優位が鮮明になる。{{< /point >}}

### 地方転職の成功事例【具体的な数字あり】

「理論はわかった。でも実際にうまくいった人はいるの？」正直なところ、そう思いますよね。

具体的な転職成功事例を紹介します。

事例1：Aさん（27歳女性、広島県）
病院薬剤師から調剤薬局に転職。年収350万円→481万円（+131万円）。職場の規模と環境が年収に大きく影響することを実感したと言います。

事例2：Bさん（27歳男性、北海道→東京）
公務員薬剤師から調剤薬局に転職。年収400万円→600万円（+200万円）。業種を変えるだけでこれだけ変わった事例です。

事例3：Mさん（転職歴約20年）
中規模調剤薬局から転職。業務内容はほぼ変わらず年収60万円アップ。転職前に市場相場を調べることの重要性を強調されていました。「同じ仕事をしているのに、同僚と給料が違った」という気づきが転職のきっかけだったそうです。

{{< box color="cyan" title="派遣薬剤師で450万円→984万円アップした事例（参考）" >}}
正社員転職以外に「派遣薬剤師」という選択肢もあります。派遣は時給が高く、年収450万円→984万円へ短期間でアップした事例も。ただし、ボーナスなし・雇用が不安定といったデメリットもあるため、ライフプランを踏まえて検討してください。
{{< /box >}}

{{< point >}}成功事例に共通するのは「市場相場を調べてから動いた」こと。転職エージェントを使って相場感を身につけることが第一歩。{{< /point >}}

### 職場見学・見学時のチェックポイント

「面接では聞きにくいことが、見学に行けばわかる」というのが現場の実感です。

職場見学では、以下のポイントを必ず確認してください。

1. 処方箋の枚数・在宅訪問件数（1日・1週間あたり）
2. スタッフの人数と年齢構成（1人でほぼ全部やる薬局は要注意）

{{< point >}}在宅転職の見学では「訪問時間帯」と「スタッフ人数」を必ず確認する。ここを見れば実態がわかる。{{< /point >}}

### 条件交渉で損しないための交渉術

### よくある失敗パターンと回避法

「リスクも正直に教えてほしい」という方に、包み隠さずお伝えします。

転職相談で実際に聞こえてくる失敗パターンと、その対策をまとめます。

{{< box color="red" title="要注意：よくある失敗パターン3選" >}}
- 失敗1：大手調剤薬局→個人薬局に転職して給与が大幅低下
- 失敗2：求人票には高時給と書いてあったが、ボーナスなしで年収は低かった
- 失敗3：高年収求人に飛びついたが、交通の便が悪く生活が不便だった
{{< /box >}}

それぞれの対策を説明します。

失敗1への対策：個人薬局は給与水準にバラつきが大きい。転職前に複数の求人を比較して相場感を持つことが重要です。「ここだけ特別に高い」求人は理由を必ず確認してください。

失敗2への対策：年収は「月給×12」ではなく「総年収（給与+ボーナス+手当）」で比較する。面接前にボーナス支給の有無・昇給制度を必ず確認してください。

失敗3への対策：転職前に現地を1〜2回は訪れて生活イメージを確認する。特にファミリーがいる場合は、学校・病院・スーパーの有無を事前に調べてください。

{{< point >}}失敗を防ぐ共通の解決策は「事前の情報収集」。エージェントを使って求人の実態を確認してから動くのが鉄則。{{< /point >}}

### 移住前に確認すべき生活インフラ・医療環境

地方移住を考えている方に特に伝えたいポイントです。

「高年収求人に飛びついて後悔」という事例の多くは、転職先の仕事ではなく「生活環境のミスマッチ」が原因です。

移住前に確認すべき項目をリストアップします。

1. 交通手段：車なしでの生活は可能か（都市部からのダウングレードに注意）
2. 医療機関：かかりつけ医・大きな病院までのアクセス
3. 教育環境：子どもがいる場合、学校・塾・習い事の選択肢
4. 商業施設：スーパー・コンビニ・ショッピングセンターの有無
5. インターネット環境：光回線・携帯電話の電波状況
6. 自治体の移住支援：移住補助金・住宅支援制度の有無（自治体によって充実度が異なる）

自治体によっては移住者向けの補助金や住宅支援が充実している場合があります。転職と移住をセットで考える際は、自治体の支援制度も調べてみてください。
