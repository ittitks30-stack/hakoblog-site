ittiさん、ちょっと聞いてもいいですか？転職エージェントって**本当に無料**なんですか？タダって聞くと逆に不安で…裏でお金取られたりしないですよね？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
よくある質問ですね。**結論から言うと、求職者は1円も払いません**。登録から内定まで、面談・求人紹介・書類添削・面接対策・年収交渉まで全部無料です。

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}

{{< /chat >}}

## エージェントは「企業」からお金をもらっている

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
お金を払っているのは**採用する企業側**です。エージェント経由で求職者を採用したら、企業は成功報酬としてエージェントに<strong>理論年収の30〜35%</strong>を支払うのが相場ですね。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
**30〜35%！？** 例えば年収500万円の人が転職したら…
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
企業はエージェントに**150〜175万円**払う計算です。だからエージェントは「年収の高い人を、なるべく早く転職させたい」というインセンティブが働くわけですね。
{{< /chat >}}

{{< box color="cream" title="お金の流れまとめ" >}}
- 求職者（あなた） → エージェント：**0円**
- 企業 → エージェント：<strong>理論年収の30〜35%</strong>（成功報酬）
- 採用が決まらなければ企業も支払いゼロ（完全成功報酬型）
{{< /box >}}

## なぜ企業は高いお金を払ってまで使うのか

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
でも企業からしたら150万円って高くないですか？自社で求人サイトに載せたほうが安いんじゃ…
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
鋭いですね。実は企業側にもメリットがあるんです。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
**1つ目は「採用工数の削減」**。求人広告だと数百人の応募が来て書類選考だけで人事が疲弊しますが、エージェント経由なら事前にフィルタリング済みの候補者だけ来ます。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
**2つ目は「ミスマッチ防止」**。エージェントが間に入って求職者と企業の希望をすり合わせるので、入社後すぐ辞めるリスクが下がります。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
**3つ目は「非公開採用ができる」**。新規事業の責任者や役員クラスなど、表に出せないポジションは100%エージェント経由ですね。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
なるほど！企業にとっても<strong>「広告より安く済む」「いい人だけ来る」</strong>というメリットがあるから、150万円払う価値があるんですね。
{{< /chat >}}

## 無料には「3つの落とし穴」がある

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
じゃあ無料で安心して使っていいってことですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
基本はそうですが、**無料ゆえの落とし穴も知っておいてほしい**です。
{{< /chat >}}

### ① 利益にならない求職者は後回しにされる

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
エージェントは成功報酬ビジネスなので、**「転職が決まりそうな人」を優先**します。希望条件が市場とズレすぎていたり、ブランクが長かったりすると、対応が雑になることも残念ながらあります。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
ちょっとシビアですね…私は大丈夫かな？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
さくらさんは28歳・営業経験ありなので**売り手市場の典型的な層**ですよ。むしろ複数社から取り合いになるはずです。
{{< /chat >}}

### ② 早く決めさせたいバイアスがかかる

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
報酬は「決まってナンボ」なので、**早く内定承諾させたいインセンティブ**が働くこともあります。「他にもいい案件出ますよ」とじっくり待つより、「いま決めたほうがいいですよ」と背中を押されがちです。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
「今日中に返事ください」とか言われたら、ちょっと怪しんだほうがいいですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
基本は2〜3日は待ってもらえます。**極端に急がせる担当者は要注意**ですね。
{{< /chat >}}

### ③ 紹介求人がエージェントの都合に偏ることがある

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
**「決まりやすい求人」「報酬の高い求人」を優先的に紹介**されるケースもあります。本当にあなたに合った求人が他にあっても、紹介リストに載ってこないことも。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
それを防ぐにはどうすれば？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
**複数のエージェントに登録して、紹介求人を比較する**のが一番確実です。1社だけだと比較対象がないので、その求人が良いのか悪いのか判断できません。
{{< /chat >}}

{{< box color="yellow" title="無料でも安心して使う3つのコツ" >}}
1. **複数登録**して紹介求人を比較する（最低2〜3社）
2. **急かす担当者**には「他社とも比較したい」とハッキリ伝える
3. **希望条件は具体的に**伝えて、優先度の高い求職者として扱われる準備をする
{{< /box >}}

## 後から請求が来ることはないの？

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
本当に**後からお金請求されたりしない**ですよね？転職してから半年後に「実は…」みたいな…
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
**100%ありません**。職業安定法で「求職者から手数料を徴収してはならない」と法律で禁じられています（一部の特殊職種を除く）。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti.png" color="blue" >}}
もし後から請求が来たら、それは**違法業者**です。すぐに労働局や消費生活センターに相談してください。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura.png" color="pink" >}}
法律で守られてるんですね。それなら安心して使えそう！
{{< /chat >}}

{{< box color="green" title="この記事のまとめ" >}}
- 転職エージェントは**求職者からは完全に無料**（法律で守られている）
- お金は**企業が成功報酬で支払う**（年収の30〜35%）
- ただし「決まりやすい人優先」「早く決めさせたいバイアス」など**無料ゆえの注意点**もある
- 対策は**複数登録して紹介求人を比較する**こと
- まず大手1社＋特化型で2〜3社の登録から始めよう
