転職・キャリア支援に10年以上携わってきたittiが、女性向けキャリアコーチングの2大サービス「LadyCarry」と「ZaPASS」を実際に調べ・比較した内容をまとめました。

この記事でわかること：

- LadyCarryとZaPASSの料金・特徴・口コミの違い

## 女性向けキャリアコーチングが注目される背景

### 女性特有のキャリア課題とは

女性管理職の割合は2023年度時点で12.7%（厚生労働省「令和5年度雇用均等基本調査」）。政府目標の30%にはまだ遠い状況です。帝国データバンクの調査でも平均10.9%と「初の1割超え」が話題になりましたが、それでも3割にはほど遠い。

マイナビの調査によると、女性は**30代で「昇進したくない」が「昇進したい」を上回る**という特徴があります。男性は40代での逆転なので、10年早い転換点です。昇進を望まない理由のトップは「責任が重くなるため（49.2%）」で、次いで「自分には向いていない（39.8%）」「家庭との両立が困難（16.7%）」と続きます。

「給料は低いままで昇進の話もこない。女性だからなのかスキルがないのかわからない」。この声、正直なところ、データの裏側がそのまま出ている感じがします。

育休復帰の局面でも、73.4%のママが産休・育休後のキャリアに「不安を感じる」というデータがあります（株式会社明日香、2024年）。子育てと仕事の両立が心配（82.5%）、収入面の不安（68.8%）が主な理由です。「育児退職」を経験した女性は5人に1人にのぼります（マイナビ、2024年）。

{{< box color="blue" title="データで見る女性のキャリア不安" >}}
- 女性管理職比率：12.7%（政府目標は30%）
- 女性は30代で「昇進したくない」が逆転する特有パターン
- 73.4%のママが育休後キャリアに不安を感じている
- 育児退職経験は女性の5人に1人
{{< /box >}}

{{< point >}}女性のキャリア課題には構造的・社会的な背景があり、一人で抱え込む問題ではない。{{< /point >}}

### 転職エージェントとの違いを理解する

「転職エージェントに相談したら求人を紹介されるだけで、自分がどうしたいかを考える時間がなかった」という声をよく聞きます。分かります、その感覚。

転職エージェントは「転職・内定獲得」という結果を目的とした支援です。一方、キャリアコーチングは「自分が何をしたいか・どう生きたいか」を掘り下げる支援。目指すゴールが根本的に異なります。

| | 転職エージェント | キャリアコーチング |
|---|---|---|
| 目的 | 転職・内定獲得 | 自己理解・意思決定 |
| サービス | 求人紹介・書類添削 | セッション・内省支援 |
| 費用 | 無料（企業負担） | 有料（数十〜百万円台） |
| 期間 | 数週間〜3ヶ月 | 3〜6ヶ月 |
| 結果 | 転職できる | 自分に合う選択ができる |

キャリアコーチングでは「転職しない」という結論も立派な成果です。現職残留・副業・起業など、選択肢を広げた上で自分の意思で選べるようになることが目的です。

## LadyCarryとは？特徴・料金・口コミを検証

### サービス概要とコーチの特徴

LadyCarryは、女性専用のキャリアコーチングサービスです。「適職を見つけて自分らしく輝ける女性を増やす」をコンセプトに掲げています。

最大の特徴は、コーチが全員女性であること。結婚・出産・育児といったライフイベントを「特別な事情」ではなく「前提」として扱います。「男性コーチには言いにくい話がある」という方には、この安心感は大きいですよね。

累計相談者数1,000人以上、顧客満足度98%、内定率100%を公式発表しています（自社発表の数値のため参考値としてご覧ください）。anan・日経WOMANへのメディア掲載実績もあります。

{{< box color="green" title="LadyCarryの特徴まとめ" >}}
- 全員女性コーチ（ライフイベント経験者）
- 累計相談者数1,000人以上・顧客満足度98%
- 完全オンライン（ZOOM）
- 無料カウンセリング60分あり（2,000名以上参加実績）
- anan・日経WOMANに掲載
{{< /box >}}

### 料金プランと利用の流れ

LadyCarryの利用の流れは以下のとおりです。

1. 無料カウンセリング（60分、ZOOM）← ここで相性を確認できる
2. コースを選択・契約
3. コーチとのセッション開始（ZOOM）
4. 自己分析・キャリア設計・行動計画
5. 転職・昇進・現職での新たなアクション

注意点として、**入会金55,000円が表示価格に加えて別途必要**です。他サービスと比較するときは、入会金込みの実質額で計算するようにしましょう。

## ZaPASSとは？特徴・料金・口コミを検証

### サービス概要とコーチの特徴

ZaPASS（ザッパス）は「完全カスタマイズ型オンラインパーソナルコーチング」を掲げるサービスです。女性専用ではなく、キャリアだけでなく人生・目標全般を対象にしています。

コーチ陣は約200名。大手企業のマネジメント経験者・スタートアップ経営者・ベンチャー事業責任者など、豊富なビジネス経験を持つ人材が揃っています。「ビジネス視点でキャリアを整理してほしい」という方には、この多様性はありがたいですよね。

特徴的なのが「ZaPASSエゴグラム診断」です。感覚ではなく性格・好み・目的に基づく科学的なマッチングでコーチを選定します。顧客満足度は10点満点中8.4点（体験顧客調査）、入会金が不要なのも大きなポイントです。

{{< box color="green" title="ZaPASSの特徴まとめ" >}}
- 約200名の多様なコーチ（男女・多業種）
- エゴグラム診断による科学的マッチング
- 全プランLINEサポート標準装備
- 入会金なし
- 顧客満足度8.4/10.0
{{< /box >}}

### 料金プランと利用の流れ

ZaPASSの利用の流れです。

1. 無料カウンセリング（60分）← エゴグラム診断もここで実施
2. コースを選択・担当コーチが決定
3. セッション開始（オンライン、週次）
4. LINEで日常的なサポートを受けながら継続
5. 中長期キャリアの方向性を確立

料金は以下のとおりです。入会金がないため、表示価格がそのまま実質価格になります。

| プラン | 料金 | 期間 | セッション数 | 特徴 |
|---|---|---|---|---|
| マインド変容プラン | 495,000円 | 90日 | 7回 | 自己認知・思考変容 |
| 行動コミットプラン | 660,000円 | 180日 | 10回 | 中長期行動設計 |
| 行動加速プラン | 770,000円 | 240日 | 10＋3回 | 強み深化＋加速 |

全プランでLINEサポートとエゴグラム診断が標準でついています。

注意点として、ZaPASSは求人紹介なし・転職急行非対応・技術スキルアップは対象外です。あくまで「思考・行動パターンの変容」に特化したサービスです。

{{< point >}}ZaPASSは入会金なし。LINEサポートが標準装備で日常的なフォローが手厚い。{{< /point >}}

公務員からIT転職を果たした女性の体験談も注目に値します。「ネガティブ思考が多かったが、不安と自分を切り離して考えることを学んだ」という変化は、コーチングならではの成果でしょう。

- LadyCarry最安：396,000 + 55,000 = 451,000円（80日・7回）
- ZaPASS最安：495,000円（90日・7回）

「LadyCarryの方が安い」と思いがちですが、入会金を含めた差額は約44,000円。期間・セッション数が近いことを考えると、実質的にほぼ同水準です。
