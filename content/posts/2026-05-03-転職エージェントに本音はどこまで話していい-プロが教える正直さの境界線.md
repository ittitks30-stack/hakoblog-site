「転職エージェントには本音で話したほうがいい」とよく聞くけど、**前職の不満や年収のホンネ、退職理由まで全部正直に話していいの？** ――今日は転職コンサル歴10年の **itti** が、転職活動2回目の **さくらさん（28歳・営業職）** の疑問にズバッと答えていきます。

ittiさん、今度エージェントとの面談があるんですけど…正直、**前職の人間関係でめちゃくちゃ揉めて辞めた**んですよ。これって正直に話したほうがいいんですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
よくある悩みですね。結論から言うと、**「事実は話す、感情は控える」が黄金ルール**です。「人間関係で揉めた」より「チームの方向性が合わず、新しい環境を求めた」のように"昇華"して伝えるのがベストです。

## なぜ本音で話したほうがいいのか

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
そもそも、なんで本音で話したほうがいいんでしょう？多少盛ったほうが良い印象になりそうな気もして…
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
理由は2つあります。**1つは「ミスマッチ防止」**。残業を本当は嫌なのに「全然OKです」と言うと、また同じような会社に紹介されて、入社後に苦しむことになります。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
**2つ目は「企業に伝わる情報の正確性」**。エージェントは推薦コメントを書いて企業に送ります。表面的な情報だけだと、面接で深掘りされたときにボロが出やすい。事実ベースの推薦のほうが内定率が上がります。
{{< /chat >}}

{{< box color="cyan" title="本音を話したほうがいい3大理由" >}}
- **ミスマッチ防止**：希望と違う求人を紹介されない
- **推薦の説得力UP**：エージェントが企業に送る推薦が具体的になる
- **面接対策の精度UP**：あなたの本当の弱みに合わせた対策をしてもらえる
{{< /box >}}

## 「話していいこと」と「控えるべきこと」の境界線

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
じゃあ何でも正直に話していいんですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
いえ、**「事実」は話す、「感情」「悪口」「他社情報」は控える**が基本ルールです。表でまとめましょう。
{{< /chat >}}

| 内容 | 本音で話すべき | 表現を工夫すべき |
|---|---|---|
| 希望年収・勤務地 | ◎ 必須 | – |
| 退職理由（事実） | ◎ 話す | – |
| 前職の不満（感情） | △ | 「環境を変えたい」など昇華 |
| 上司・同僚の悪口 | × | 言わない |
| 健康状態（業務影響あり） | ◎ 話す | – |
| 持病（業務影響なし） | △ | 任意 |
| 副業・SNSアカウント | △ | 業務関連のみ |
| 他社の選考状況 | ◎ 話す | – |
| 他社エージェントの担当者名 | × | 言わない |

## ① 退職理由：事実は伝える、感情は昇華する

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
じゃあ私の「人間関係で揉めて辞めた」はどう伝えればいいですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
**事実**：「営業方針について上司と意見の相違があった」「裁量を持って仕事をしたいと考えるようになった」これは話してOK。<br>
**NG**：「上司がパワハラ気味で」「同僚が陰口ばかり言う」これは絶対に言わない方がいい。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
理由はシンプルで、**愚痴を言う人は採用側からすると「次の職場でも同じことを言うかも」と思われる**から。エージェントにすら愚痴ってしまう人だ、と判断されます。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
うっ…耳が痛いです。気をつけます。
{{< /chat >}}

## ② 年収：希望は強気で言う、現年収は正直に

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
年収って、現年収を盛っちゃダメですよね？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
**現年収はバレます**。源泉徴収票の提出を求める企業がほとんどなので。逆に**希望年収は強気で言ってOK**。「希望550万、最低450万」のように幅を持たせるのがコツです。
{{< /chat >}}

{{< box color="yellow" title="年収交渉でやってはいけない3つ" >}}
1. 現年収を盛る → ほぼ確実にバレる（源泉徴収票でアウト）
2. 希望年収を低く言う → エージェントの紹介求人レンジが下がる
3. 「いくらでもいいです」 → 足元を見られて低単価求人を回される
{{< /box >}}

## ③ 他社の選考状況：必ず共有する

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
他のエージェント経由で選考が進んでる会社があるんですけど…これも話していいんですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
**会社名や進捗は積極的に話してください**。「A社が来週最終面接、B社が今月内定予定」のように。これがあると**エージェント側も内定承諾の期限調整を企業に交渉**できるんです。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
ただし、**「どのエージェント経由か」までは言わなくてOK**。エージェント同士は競合なので、不要な情報です。
{{< /chat >}}

## ④ 病気・メンタル：業務に影響ないなら任意

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
2年前に少しメンタルやられたんですけど、今は完全に治ってます。これって言うべきですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
**今の業務に支障がないなら、言う義務はありません**。逆に通院中・服薬中で配慮が必要なら、エージェントには共有して、入社前に企業へ「健康面の配慮事項」として伝えてもらう形がベストです。
{{< /chat >}}

{{< box color="gray" title="健康情報の伝え方の目安" >}}
- **完治・後遺症なし**：話さなくてOK（プライバシー）
- **通院中だが業務影響なし**：エージェントには共有、企業には任意
- **業務に配慮が必要**：エージェント・企業の両方に共有（入社後トラブル防止）
{{< /box >}}

## ⑤ 嘘がバレる5大パターン

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
逆に、よくある「これは嘘がバレる」って例ありますか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
ありますね。プロから見ると一発でわかるパターンが5つあります。
{{< /chat >}}

{{< box color="red" title="プロが見抜く嘘パターン5選" >}}
1. **現年収の盛り** → 源泉徴収票提出でアウト
2. **在籍期間の改ざん** → 社会保険記録・離職票で発覚
3. **役職・成果の盛り** → 面接で具体的な数字を聞かれて沈黙
4. **退職理由の偽装** → リファレンスチェックで前職に確認される
5. **資格・スキルの誇張** → 入社後の業務でバレて評価ダウン
{{< /box >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
特に最近は**リファレンスチェック**（前職の同僚・上司への確認）が増えています。嘘は本当にバレやすい時代ですよ。
{{< /chat >}}

## それでも本音を話しにくい時の対処法

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
担当者と相性が合わなくて、本音を話す気になれない時はどうすれば？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
**担当者を変更してもらってください**。エージェント会社に「相性的に別の方とお話したい」と伝えればOK。これは普通の手続きで、角が立つこともありません。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
それでもダメなら、**別のエージェントに乗り換える**。複数登録しておけば、相性の良い担当者と巡り合える確率が上がります。
{{< /chat >}}
