転職エージェントとの面談を前に「どこまで話していいんだろう…」と不安になっていませんか？

この記事では、転職コンサル歴10年の **itti（イッチ）** が、転職2回目の **さくら** さんの疑問にリアルに答えながら、「話すべき本音」と「伝え方を工夫すべき本音」を項目別に解説していきます。精神論ではなく、具体的な判断基準と言い換えテンプレートを使って、「戦略的な正直さ」を身につけましょう。

## 転職エージェントは「敵」ではなく「商談相手」と捉える

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
先日、転職エージェントに登録したんですが……正直に「給料が低くて不満」って言っても大丈夫なんですか？ 印象悪くなりそうで怖くて。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
その不安、すごくよくわかります。でも結論から言うと、エージェントは「敵」ではなく「商談相手」なんですね。まずエージェントのビジネス構造を知ると、なぜ正直さが大事かがわかってきます。
{{< /chat >}}

### エージェントのビジネスモデルを理解する（成功報酬型）

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
ビジネス構造？エージェントってどうやってお金をもらってるんですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
転職エージェントは「成功報酬型」です。あなたが転職を決めて入社したとき、<strong>採用した企業から手数料をもらう</strong>仕組みになっています。相場は採用者の想定年収の<strong>30〜40%</strong>。年収500万円の人材が決まれば、企業からおよそ<strong>175万円</strong>が支払われる計算です。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
えっ、そんなにもらってるんですか！? じゃあ私たち求職者は無料で使えるってことですよね？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
そうです。求職者からは1円もとりません。ここが重要なポイントで、エージェントの報酬は「あなたが満足して長く勤める会社に転職できたとき」に最大化されます。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
さらに、入社後6ヶ月以内に退職すると企業への<strong>返金規定が発動</strong>することが多い。だから、条件ミスマッチによる早期退職は、エージェント側にとっても損なんです。
{{< /chat >}}

{{< box color="blue" title="エージェントのビジネスモデル：3つのポイント" >}}
1. <strong>完全成功報酬型</strong>：採用が決まった時点で企業から手数料が発生（求職者は無料）
2. <strong>手数料相場は年収の30〜40%</strong>：年収500万円 → 企業が約150〜200万円を支払う
3. <strong>早期退職で返金リスクあり</strong>：入社後6ヶ月以内の退職は返金対象になることが多く、エージェント側もミスマッチを防ぎたい
{{< /box >}}

### 正直さがエージェントにもたらすメリット

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
つまり、あなたが「本当の希望・本当の条件」を話してくれるほど、エージェントも<strong>マッチ度の高い求人</strong>を提案しやすくなります。曖昧な情報をもとに動いても、双方にとって時間のムダ。「商談相手」として同じゴールに向かっているイメージを持ってもらえると、気持ちが楽になりますよ。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
なるほど！エージェントにとっても、私が本音を話してくれる方が都合がいいわけですね。少し安心しました。
{{< /chat >}}

**ポイント：エージェントはあなたの転職成功が自分の報酬につながる「利益を共有した商談相手」**

## 「話すべき本音」と「伝え方を工夫すべき本音」の違い

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
でも「なんでも正直に話せばいい」ってわけでもないですよね？ 人間関係が辛いとか、上司が嫌いとか、そのまま言っていいのかな……。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
鋭いです。本音には2種類あります。「そのまま話すべき本音」と「内容は正直だけど伝え方を工夫すべき本音」。この2つをきちんと区別するのが戦略的正直さの核心です。
{{< /chat >}}

### 話すべき本音①：転職の真の動機（ポジティブ変換のコツ）

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
転職の動機は、<strong>正直に伝えた上で「未来志向」の言葉に変換</strong>するのがコツです。「給与が低い」は事実として伝えてOK。ただし「低いから逃げたい」より「年収○○万円以上の環境で○○を実現したい」と未来のゴールとセットで伝える方が、エージェントも求人を絞りやすくなります。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
えっと……たとえば「残業が多くて疲弊してる」は、どう変換するんですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
「月40時間超の残業が続いており、プライベートの時間を大切にしながらパフォーマンスを上げたいため、残業月20時間以内の環境を希望しています」——こう言えば、エージェントは残業時間のデータがある求人を優先して提案してくれます。不満を「要件定義」に変換するイメージです。
{{< /chat >}}

{{< box color="green" title="ポジティブ変換テンプレート" >}}
| 本音（そのまま） | 変換後（エージェントへの伝え方） |
|---|---|
| 給料が低い | 「年収○○万円以上の環境を求めています」 |
| 残業が多すぎる | 「残業月○○時間以内の職場を優先したいです」 |
| 上司が嫌い | 「フラットな組織文化・風通しのよい職場を重視しています」 |
| 会社の将来が不安 | 「安定性・成長性のある事業領域に移りたいです」 |
| 仕事が単調でつまらない | 「より裁量のある仕事・成長機会を求めています」 |

**ポイント**：「何から逃げるか」ではなく「何に向かうか」で表現する。内容は同じでも、エージェントが動きやすくなります。
{{< /box >}}

### 話すべき本音②：年収・条件の希望（下限と理想の両方を伝える）

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
年収希望を正直に言うと、求人が絞られすぎて選択肢が減っちゃいそうで……。高い金額を言うのをためらってしまいます。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
それ、逆効果になりやすいパターンです。年収は「譲れない下限」と「理想の上限」の2つをセットで伝えるのが正解。「最低でも450万円、理想は550万円以上」のように伝えると、エージェントが幅を持って提案できます。下限を隠すと、条件の合わない求人を大量に紹介されるリスクがあります。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
実際、転職したら年収ってどれくらい変わるんですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
マイナビの「転職動向調査2025年版」によると、2024年の転職後の平均年収は<strong>509.3万円</strong>。転職前の平均487.3万円から平均で<strong>22万円増</strong>という結果が出ています。年収アップした人の割合は<strong>39.4%</strong>でした。条件を正直に伝えて交渉する姿勢が、この数字につながるんです。
{{< /chat >}}

### 伝え方を工夫すべき本音：現職への不満・人間関係の問題

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
一方で「伝え方の工夫」が必要なのは、<strong>現職への愚痴・人間関係の不満</strong>です。「上司がパワハラ気質で」「同僚と全く合わなくて」——こうした話をそのまま続けると、エージェントや面接官に「問題がある人」と誤解される可能性があります。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
でも、それが本当の退職理由なんですけど……隠すんですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
隠さなくていいです。ただ「何があったか」より「どんな環境を求めているか」にフォーカスして話しましょう。「チームの連携が難しい環境だった」→「協調しやすい職場文化を重視しています」という形です。エージェントには背景を正直に話し、面接での言い方のアドバイスをもらうのが最善策です。
{{< /chat >}}

{{< box color="yellow" title="注意：現職への不満を伝えるときの落とし穴" >}}
- <strong>前の職場の悪口に終始しない</strong>：エージェントも「この人はどこに行っても不満を言いそう」と判断することがある
- <strong>個人攻撃にならないよう抽象化する</strong>：「あの上司が」→「組織の意思決定スタイルが合わなかった」
- <strong>ただし我慢して黙る必要はない</strong>：退職理由の背景（ハラスメント・健康被害等）は正直に伝えてよい。適切な対応を一緒に考えてもらえる
{{< /box >}}

**ポイント：本音は伝える。ただし「どんな環境を求めているか」という未来志向の言葉に変換してから伝える**

## 絶対に正直に話すべき3つの項目

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
逆に「これは絶対に正直に言わないとダメ」っていう項目はありますか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
3つあります。これを隠すと、後でトラブルになるリスクが高い項目です。
{{< /chat >}}

### 健康状態・持病（隠すとトラブルの原因に）

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
持病や通院中の疾患は、エージェントには正直に話しておきましょう。「無理に配慮不要の職場に入ってしまう」「入社後すぐ欠勤が増えて早期退職になる」——こうなると本人も、エージェントも、企業も、誰も得をしません。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
精神的に少し辛かった時期があって……そういうのも話すべきですか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
エージェントには話しておくのがおすすめです。「現在は回復しているが、残業が多い職場は避けたい」といった形で、<strong>今の状態と求める環境</strong>を伝えればいい。面接でどこまで開示するかは、エージェントと一緒に戦略を立てられます。一人で抱え込まないことが大事です。
{{< /chat >}}

{{< box color="red" title="健康情報の開示：エージェントへの伝え方" >}}
<strong>エージェントに話してよいこと</strong>（転職活動のサポートに直結するため）
- 現在治療中の疾患・定期通院の有無
- 職場環境の希望（残業上限・テレワーク可否・配慮事項）
- 過去の休職歴とその後の回復状況

<strong>面接・内定後の開示判断はエージェントに相談する</strong>
- 法的には「業務遂行に直接関わる」情報のみ開示義務がある
- どの段階でどこまで話すかは、職種・会社の規模・疾患の種類によって異なる
{{< /box >}}

### スキル・経験の有無（過大申告のリスク）

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
スキルの過大申告は絶対NG。「少しだけ触ったことがある」を「経験あり」として話すのは、後に必ずバレます。入社後に「話と違う」となれば、企業側がエージェントへの手数料返金を求めるケースもあります。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
逆に、未経験だと正直に言ったら求人を紹介してもらえなくなりませんか？
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
「未経験だが○○を独学中」「資格取得に向けて勉強している」という<strong>今のリアルな状態</strong>を伝えれば、それを評価する求人に絞って提案してもらえます。エージェントはあなたを「今この状態でどこにマッチするか」で動くのが仕事ですから。
{{< /chat >}}

### 転職活動の軸・譲れない条件

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
<strong>絶対に譲れない条件</strong>は最初にはっきり言っておきましょう。「週2テレワーク必須」「転勤なし」「土日祝休み」など。これを曖昧にすると、エージェントは無難な大量の求人を送ってきます。実際、ある体験談では担当者から「最低20件は応募しましょう」と言われ、<strong>80%</strong>が希望と異なる求人だったというケースもありました。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
80%も！それは辛い……。軸をはっきり言うことで、そういう無駄な提案を減らせるんですね。
{{< /chat >}}

{{< box color="cyan" title="「転職の軸」を整理する3つの質問" >}}
1. **絶対に譲れない条件は何か？**（残業・転勤・給与・雇用形態など）
2. **できれば叶えたい希望は何か？**（業界・職種・働き方など）
3. **5年後にどうなっていたいか？**（キャリアの方向性）

この3つをメモしてから面談に臨むだけで、エージェントの提案精度が大きく上がります。
{{< /box >}}

## エージェントに言わなくていいこと・言うべきでないこと

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
逆に、言わなくていいことってあるんですか？ 「全部正直に」じゃないんですね。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
あります。戦略的正直さとは、「言うべきことを正確に言う」こと。「言わなくていいことまでわざわざ言う」ことではありません。
{{< /chat >}}

### 競合エージェントや直接応募の状況（全開示は不要）

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
複数のエージェントを使っていること自体は隠さなくて構いません。ただし「A社には○○社に応募中と言った」といった各社への具体的な開示内容まで話す必要はないです。2024年のデータでは転職者の<strong>87.3%</strong>が複数エージェントを併用しています。それが普通なんです。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
87.3%って、ほぼ全員ですね！ じゃあ複数使っていることを隠す必要はないんですね。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
そうです。「今○社に並行して登録しています」程度を伝えれば十分。各社に言っていることが矛盾しないよう、希望条件・転職軸は<strong>全エージェントで統一</strong>しておくのが重要です。エージェントAには「年収優先」、エージェントBには「ワークライフバランス優先」と言ってしまうと、紹介求人がバラバラになります。
{{< /chat >}}

{{< box color="gray" title="複数エージェントを使うときの情報管理" >}}
**統一して全社に伝えること**
- 転職の軸（譲れない条件）
- 年収の下限・理想
- 希望職種・業界
- 入社可能時期

**各社にあえて揃えなくていいこと**
- 他社の選考状況の詳細
- 各社で紹介された求人の内容
- 他エージェントへの不満・比較
{{< /box >}}

### 内定後の交渉カードになる情報

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
もう1つ。他社から内定をもらった場合の年収額・条件は、<strong>交渉の最終局面まで手の内を明かさない</strong>方が有利なこともあります。「他社で○○万円の内定が出た」という情報は、条件交渉の切り札です。エージェントには「他社内定が出ています」と伝えるにとどめ、具体的な数字は自分で判断して出すタイミングを決めましょう。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
なるほど、交渉カードは大事にするということですね。
{{< /chat >}}

**ポイント：複数エージェント利用は普通のこと。ただし転職の軸・条件は全社で統一し、矛盾した情報を流さない**

## エージェントとの信頼関係を築く実践的なコツ

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
でも実際、エージェントって全員が信頼できるわけじゃないですよね……。「内定確実ですよ！」と言ったのに落ちたとか、求人内容と実際が全然違ったとか、ネットで体験談を見てちょっと怖くて。
{{< /chat >}}

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
正直に言うと、残念ながらすべてのエージェントが誠実とは限りません。だから最初の面談で「この担当者は信頼できるか」を見極めることが大切です。
{{< /chat >}}

### 最初の面談で「信頼できるか」を見極める質問

{{< chat name="itti" side="left" avatar="/images/chars/itti_anime.png" color="blue" >}}
面談で担当者に投げかけると、信頼度が見えやすい質問があります。「この会社に応募するデメリットも教えてください」と聞いてみてください。メリットしか話さない担当者は要注意。デメリットも正直に伝えてくれる担当者は、長い目で見てあなたの利益を優先してくれている可能性が高いです。
{{< /chat >}}

{{< chat name="さくら" side="left" avatar="/images/chars/sakura_anime.png" color="pink" >}}
なるほど！ それは面白い見極め方ですね。他にチェックポイントはありますか？
