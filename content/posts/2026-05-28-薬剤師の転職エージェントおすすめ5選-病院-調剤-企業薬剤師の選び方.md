「どの転職エージェントを選べばいいか、正直全然わからない…」

そう感じている薬剤師の方、とても多いです。

現職の給与に不満があっても、人間関係で消耗していても、いざ転職エージェントを探してみると選択肢が多すぎて混乱してしまいますよね。分かります。

この記事では、薬剤師転職に関する記事を50本以上執筆・調査してきたittiが、以下の悩みをわかりやすく解説します。

- どのエージェントが自分の希望職種（病院・調剤・企業）に強いのか
- 非公開求人にアクセスする方法
- エージェントのしつこい連絡を穏やかに断る具体的な方法

この記事を読めば「自分にはどのエージェントが合うか」が具体的にわかり、登録後に後悔しない選択ができます。

**まず結論として、薬剤師転職エージェントは2〜3社の複数登録が正解です。** 理由は後半で詳しく説明します。

## 薬剤師が転職エージェントを使うべき理由

### ① 一般求人サイトとの決定的な違い

「求人サイトで自分で探せばいいのでは？」と思う方もいるかもしれません。

正直なところ、薬剤師の転職においては求人サイトだけで動くのはかなりもったいないです。なぜかというと、転職エージェントには非公開求人が存在するからです。

ようするに、一般公開されていない求人のことです。条件の良い求人や競争率が高い求人は、あえて非公開にして質の高い求職者に絞って紹介するケースが多い。

エージェントを使う主なメリットをまとめると：

- 非公開求人へのアクセス（全求人の40〜80%が非公開のケースも）
- 給与・条件交渉を代行してもらえる
- 職場の内部情報（人間関係・残業実態など）を事前に把握できる
- 書類添削・面接対策のサポートがある

「給与交渉など自分では切り出しにくいことを代わりにやってくれた」という声は非常に多いです。40代後半の女性が病院から調剤薬局へ転職した際の体験談でも、担当者が交渉の席に同席してくれたことで内定条件が大幅に改善したケースがあります。

{{< box color="blue" title="転職エージェントが無料で使える理由" >}}
薬剤師向け転職エージェントは、採用した企業側から紹介手数料を受け取るビジネスモデルのため、求職者（薬剤師）は完全無料で使えます。費用は一切かかりません。
{{< /box >}}

{{< point >}}エージェントは無料・非公開求人へのアクセス・交渉代行という3つの価値を持つ{{< /point >}}

---

### ② 非公開求人の重要性

薬剤師市場では、非公開求人の割合が高いエージェントほど質の良い求人を保有している傾向があります。

例えば、後述するアポプラス薬剤師は全求人43,537件のうち約8割が非公開求人です。これだけ見ても、エージェントに登録しないと見えてこない求人の多さがわかります。

非公開求人が多い理由は主に3つあります。

1. 競合他社に採用活動を知られたくない（大手調剤薬局チェーンなど）
2. 応募が殺到して選考が混乱するのを防ぎたい
3. 特定のスキル・経験を持つ人材だけにアプローチしたい

非公開求人は、エージェントに登録して担当者と面談してから初めて紹介してもらえます。つまり、エージェント登録＝非公開求人解禁、という理解で問題ありません。

{{< point >}}非公開求人の大半はエージェント経由でしか見られない{{< /point >}}

## 転職エージェント選びの3つのポイント

薬剤師向けエージェントを選ぶとき、求人数だけで判断するのは危険です。

「求人数より担当者の質のほうが転職成功率に直結する」という現実があります。以下の3点を意識して選びましょう。

### ① 薬剤師専門かどうか

薬剤師は医療職の中でも特殊な資格職です。調剤報酬の仕組み、薬局の経営構造、病院内の薬剤師の立ち位置、企業の医薬品規制対応（GMP・GQP）など、業界特有の知識がなければ適切な求人紹介はできません。

リクルートエージェントや dodaのような総合型エージェントでも薬剤師の求人は扱っていますが、担当者が医療業界に詳しいとは限りません。薬剤師専門のエージェントを軸にすることを強くおすすめします。

{{< box color="yellow" title="総合型エージェントを使う場合の注意点" >}}
総合型エージェントは求人の幅は広いですが、薬剤師業界の細かい情報（職場の人間関係・実残業時間・調剤体制）を把握していないことが多いです。薬剤師専門エージェントと組み合わせて使うのがベストです。
{{< /box >}}

### ② 希望職種の求人数

「病院薬剤師に転職したい」「企業（製薬・CRO）に行きたい」「調剤薬局でゆったり働きたい」──それぞれで強いエージェントが異なりますよね。

後半の「職種別おすすめエージェント」セクションで詳しく解説しますが、簡単にまとめると：

- 病院薬剤師希望 → 大手×専門エージェントの組み合わせ
- 調剤薬局希望 → ファルマスタッフ・アポプラス薬剤師
- 企業（研究・MR・管理薬剤師）希望 → マイナビ薬剤師

### ③ サポート体制（担当者の質・面接対策）

ここが最も重要で、かつ事前に見極めが難しいポイントです。

「担当者の当たり外れ問題」は構造的な課題として存在します。どのエージェントでも担当者によってサービスの質が大きく変わるのが現実です。

**良い担当者の見極め方：**

- 最初の面談で「転職の軸は何か」をしっかりヒアリングしてくれる
- 条件だけでなく職場環境・人間関係の情報も提供してくれる
- 「希望と少し違う求人」を理由付きで提案してくれる（視野を広げてくれる）
- 断ったときに理由を聞いて改善した提案をしてくれる

**注意すべき担当者のサイン：**

- 最初から大量の求人を無差別に送ってくる
- 転職理由を深掘りせずすぐに求人紹介に移る
- 「今すぐ決めないと枠が埋まります」と急かす

合わないと感じたら、担当者変更を遠慮なく申し出てください。これは完全に正当な依頼です。

{{< point >}}担当者の質を見極め、合わなければ変更依頼を{{< /point >}}

### 1位：ファルマスタッフ｜調剤薬局転職のデファクトスタンダード

運営会社：株式会社メディカルリソース（日本調剤グループ）
転職支援実績：22年以上
全国拠点：12拠点

ファルマスタッフが1位の最大の理由は「ミスマッチが圧倒的に少ない」点です。

転職相談満足度97.3%という数字は業界トップクラス。日本調剤グループという調剤薬局チェーンが母体にあるため、担当者が現場の実態を熟知しています。「給与表に書いてある以外の残業の実態」「職場のチームワークの雰囲気」など、求人票には出ない情報を提供してくれると評判です。

「業界の細かいことに詳しく、給与交渉なども任せられた」という利用者の声もあります。

正直なデメリット：転職活動のペースがゆっくりです。「早急に転職したい」「1ヶ月以内に決めたい」という方には向かないことがあります。また、調剤薬局以外（ドラッグストア・企業）の求人は他社に劣ります。

{{< box color="green" title="ファルマスタッフがおすすめな人" >}}
- 調剤薬局への転職を希望している
- 初めての転職でゆっくり相談しながら進めたい
- 20〜30代で教育体制の整った職場を探している
- 転職失敗のリスクを最小化したい
{{< /box >}}

### 2位：マイナビ薬剤師｜企業・病院転職なら外せない

運営会社：株式会社マイナビ
求人数：56,000件以上（2026年2月現在）
全国拠点：14〜15拠点

「製薬会社に転職したい」「病院薬剤師として大学病院に入りたい」という方には、マイナビ薬剤師が最有力候補です。大手マイナビグループのブランド力と企業ネットワークを活かした求人ラインナップが強みです。

調剤薬局以外の職種、特に企業（MR・研究・管理薬剤師・DX・医薬品安全管理）の非公開求人が充実しています。

若手・新卒向けの求人も18,000件以上と多く、「初めての転職」の方にも安心感があります。対面面談は全国15拠点で対応可能です。

正直なデメリット：電話連絡がしつこいという口コミが一定数あります。サポートの手厚さの裏返しでもありますが、気になる方は登録時に「連絡はメールで」と一言伝えることで対処できます（後述）。

{{< box color="green" title="マイナビ薬剤師がおすすめな人" >}}
- 製薬会社・企業への転職を希望している
- 病院薬剤師として大学病院・総合病院を目指している
- 20〜30代で初めての転職
- 対面面談でしっかり相談したい
{{< /box >}}

### 3位：薬キャリAGENT（エムスリーキャリア）｜スピードと年収アップを両立

運営会社：エムスリーキャリア株式会社（m3.comグループ）
求人数：60,000件以上

「薬剤師の半数が使っている」とも言われる規模感を誇る薬キャリAGENT。最大の特徴は対応スピードです。最短即日・最大10件の求人紹介が可能で、「来月から転職したい」という方にも対応できます。

また、年収600万円以上のハイクラス求人が豊富。「今の調剤薬局から年収を大幅にアップしたい」という方には特に力を発揮します。

正直なデメリット：対面面談がなく、電話・オンライン対応のみです。細かいニュアンスが伝わりにくいこともあります。「薬キャリは電話がしつこくてうざかった」という率直な口コミも存在するため、登録時に連絡方法を指定することをおすすめします。

{{< box color="green" title="薬キャリAGENTがおすすめな人" >}}
- 急いで転職を決めたい（1〜2ヶ月以内）
- 年収600万円以上のハイクラス転職を狙いたい
- 地方在住で対面面談が難しい
- すでに転職先の方向性が決まっている
{{< /box >}}

### 4位：アポプラス薬剤師｜ブランク・育児両立に強い

運営会社：クオール株式会社（調剤薬局全国チェーン）
求人数：43,537件（うち非公開約80%）

クオール薬局グループが運営するため、調剤薬局の現場知識は業界屈指。ブランク歓迎の求人が豊富で、育児との両立を考えるママ薬剤師向けの専門コンテンツも充実しています。

全求人の約8割が非公開求人という点も大きな強みです。一般には出回らない良質な求人を、面談を通じて紹介してもらえます。

正直なデメリット：ドラッグストア・OTC併設薬局を希望する場合は選択肢が少ないです。首都圏以外のエリアでは求人数が少ない地域もあります。

{{< box color="green" title="アポプラス薬剤師がおすすめな人" >}}
- 育児との両立・時短勤務を希望するママ薬剤師
- ブランク（産休・育休・病気療養）明けで復帰を考えている
- 調剤薬局・病院への転職を希望している
- 勤務時間の交渉を代行してほしい
{{< /box >}}

### 5位：ファーマリンク｜首都圏で「現役薬剤師目線」のサポートを受けたい人向け

運営会社：薬樹株式会社グループ（首都圏約147店舗）
求人数：約4,000件（うち東京都内1,200件）

他の4社と比べると求人数は少ないですが、ファーマリンクには他にない強みがあります。カウンセラー全員が薬局勤務経験を持つ現役薬剤師という点です。ようするに、薬剤師がキャリア相談に乗ってくれるということです。

「調剤スキルとコミュニケーション能力の両面でマッチングを評価する」という独自の視点で、ミスマッチを防ぐ取り組みをしています。認定薬剤師の養成サポートもあり、資格取得を視野に入れている方にも向いています。

正直なデメリット：首都圏特化のため、地方在住の方には選択肢がほぼありません。求人数も大手に比べて少ないです。

{{< box color="green" title="ファーマリンクがおすすめな人" >}}
- 東京・神奈川・千葉・埼玉での転職を希望している
- 薬剤師目線のキャリア相談を受けたい
- 認定薬剤師の資格取得も考えている
- 派遣・紹介予定派遣など柔軟な働き方を検討している
{{< /box >}}

### 病院薬剤師を目指す人向け

「病院薬剤師は給与が安い」というイメージがありますよね。正直、調剤薬局と比べると平均年収で約60万円の差があります（調剤薬局583.8万円 vs 病院薬剤師521.7万円）。

ただし、病院薬剤師には調剤薬局では得られないキャリアがあります。注射剤の無菌調製、薬剤部主任・副薬剤部長へのキャリアパス、専門薬剤師（がん・感染症・精神科など）の資格取得環境などです。

おすすめ組み合わせ：**マイナビ薬剤師 × 薬キャリAGENT**

- マイナビ薬剤師：大学病院・急性期病院の非公開求人が豊富
- 薬キャリAGENT：スピーディーな紹介でチャンスを逃さない

{{< box color="cyan" title="病院薬剤師転職のポイント" >}}
病院薬剤師は求人数が少なく競争率が高いため、エージェントへの早めの登録が重要です。大学病院の採用は「欠員補充型」が多く、タイミングを逃すと次の募集まで1〜2年待つケースも珍しくありません。
{{< /box >}}

おすすめ組み合わせ：**ファルマスタッフ × アポプラス薬剤師**

- ファルマスタッフ：調剤現場の実態把握に長け、ミスマッチが少ない
- アポプラス薬剤師：非公開求人80%、ブランク・育児両立求人が豊富

「転職したら思っていた職場と違った」という後悔を防ぐには、担当者から職場の内部情報をどれだけ引き出せるかが鍵です。両エージェントとも対面面談ができるので、細かい希望をしっかり伝えられます。

## 転職エージェントの賢い使い方・注意点

### 複数登録（2〜3社）が正解な理由

「エージェントは1社に絞るべきか、複数登録すべきか？」という疑問をよく受けます。

**答えは複数登録（2〜3社）が正解です。**

理由は3つあります。

1. エージェントごとに持っている非公開求人が異なる。1社登録では全体の求人の一部しか見えない
2. 担当者の質のバラつきリスクをヘッジできる。1社の担当者が合わなくてもカバーできる
3. 比較することで希望条件が明確になる。複数の提案を見ることで「自分が本当に求めているもの」が見えてくる

実際、「ファルマスタッフ（調剤薬局の深い情報）×薬キャリAGENT（スピードとハイクラス）×マイナビ薬剤師（企業求人）」の3社同時登録は、多くの薬剤師転職のプロが推奨する黄金の組み合わせです。

注意点として、登録しすぎると管理が大変になります。4社以上は担当者の連絡に追われてストレスになるため、2〜3社を目安にしてください。

{{< box color="yellow" title="複数登録時の管理術" >}}
- 登録エージェント名・担当者名・連絡先をメモしておく
- 「他社も登録しています」と正直に伝えるとエージェントも丁寧に対応してくれる
- 内定が出たら速やかに他社へ連絡する（誠実な対応が大切）
{{< /box >}}

### しつこい電話の断り方【具体的なメール・LINE文例付き】

薬剤師転職エージェントに登録すると、電話連絡が多くて困るという声は非常に多いです。

「仕事終わりに電話で転職の話をされて体力的にしんどかった」「判断を急がせるような連絡が多く、ストレスを感じました」というリアルな声があります。

なぜしつこいのかというと、担当者に成約ノルマがあり、早く転職が決まるほど評価が上がる仕組みだからです。ようするに、担当者を急かしているのはエージェント会社の営業体制であって、担当者個人が悪いわけではないことも多いです。

**対処法：登録直後に「連絡方法の指定」をするだけで8割防げる**

登録後の最初のメールに以下を付け加えるだけです。

{{< box color="gray" title="連絡方法指定メール文例" >}}
○○様

このたびは登録させていただきありがとうございます。
ご連絡は平日18時以降のメール（またはLINE）でいただけますと助かります。
電話対応が難しい状況が続くため、ご配慮いただけますと幸いです。
どうぞよろしくお願いいたします。
{{< /box >}}

それでも頻繁に連絡が来る場合は、担当者変更を依頼してください。「担当者を変えていただけますか」と一言伝えれば、エージェントはほぼ対応してくれます。

退会・個人情報削除を希望する場合は以下の手順で連絡してください。

1. 担当者にメールで「転職活動を終了しました」と伝える
2. 「個人情報の削除をお願いします」と明記する
3. 数日後に削除完了の連絡を確認する

無視・放置は絶対に避けてください。記録として残り、将来また同じエージェントを使いたいときに関係が悪化します。

{{< box color="red" title="どうしても連絡が止まらない場合" >}}
登録解除・個人情報削除を依頼しても連絡が続く場合は、各都道府県の労働局「需給調整事業部」に相談することができます。有料職業紹介事業者への苦情申告窓口が設けられています。
{{< /box >}}

{{< point >}}登録直後に「連絡はメールで」と伝えるだけでしつこい電話の大半を防げる{{< /point >}}

## 薬剤師転職エージェントによくある質問

{{< box color="gray" title="よくある質問まとめ" >}}
転職エージェントは完全無料・在職中も利用可能・担当者変更も可能です。疑問はどれも「まず登録して確認」が最短の答えになります。
{{< /box >}}

**Q. 転職エージェントは本当に無料ですか？**

はい、求職者（薬剤師）側は完全無料です。採用した企業が紹介手数料を支払う仕組みのため、費用は一切かかりません。

**Q. 在職中でも転職エージェントに登録できますか？**

もちろんです。むしろ在職中の転職活動が一般的です。担当者に「在職中で、連絡は○時以降でお願いします」と伝えておけば、配慮してもらえます。

**Q. 転職活動にどのくらい時間がかかりますか？**

薬剤師の場合、登録から内定まで平均1〜3ヶ月が目安です。「夏から活動して約1ヶ月で内定を獲得できた」という事例もありますが、大学病院・管理薬剤師など条件が特殊な場合は半年以上かかることもあります。

**Q. 転職後の年収はどのくらい上がりますか？**

厚生労働省「令和6年賃金構造基本統計調査」によると、薬剤師の全国平均年収は599.3万円（前年比+21.4万円）です。エージェントによる給与交渉で「求人票より20〜50万円アップした」という事例は珍しくありません。

**Q. 担当者が合わないと感じたら？**

遠慮なく担当者変更を依頼してください。「担当者の変更をお願いできますか」とメールで伝えるだけで対応してもらえます。合わない担当者のまま続けても、良い求人紹介は期待できません。

**Q. 新卒でも転職エージェントは使えますか？**

はい。マイナビ薬剤師には18,000件以上の新卒応募可能求人があります。「新卒の約8人に1人が入社1年以内に転職」というデータもあり、初転職でのエージェント活用は一般的です。
