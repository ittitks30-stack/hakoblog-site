「転職したいけど、何から始めればいいのか分からない……」

検索すると自己分析・エージェント登録・書類作成と情報は出てくるのに、どれを先にやるべきかが見えない。初めての転職では、ほとんどの人がここでつまずきます。

転職エージェントを利用して複数社から内定を得た経験のある運営者ittiが、自分の失敗も含めた実体験をもとに、「やるべきこと」を正しい順番に整理しました。

この記事でわかること：

- 転職初心者がやるべき7つのことと、その正しい順番
- 在職中でも無理なく進められるスケジュールの目安
- 初めての転職で失敗しないためのサービスの選び方

遠回りしないための道順を、順番どおりにわかりやすく解説します。

## ①転職の軸を決める｜理由と希望条件の言語化

最初にやるべきことは、求人検索でもエージェント登録でもありません。**「なぜ転職したいのか」「何を実現したいのか」という軸を言語化すること**です。

ここは私自身の失敗談から話させてください。私は「なんとなく今より良くなりたい」という曖昧な動機のまま活動を始めました。エージェントのサポートで複数社から内定をもらえたのに、最後の最後で決めきれず、結局立ち止まってしまったんです。

内定が出ること自体は、正しく動けば難しくありません。難しいのは「この会社でいいのか」を判断すること。その判断基準こそが軸です。軸がないと、せっかくの内定を前に迷子になります。

{{< box color="green" title="軸を言語化する3つの質問" >}}
- なぜ辞めたいのか（今の職場の何が課題か）を1〜2行で書く
- 転職で何を実現したいのか（年収・働き方・スキル）を書く
- 「譲れない条件」と「妥協できる条件」を分けてそれぞれ2つ挙げる
{{< /box >}}

紙に書き出すだけで構いません。この3つに答えられる状態になってから次に進むと、あとの全工程の迷いが減ります。ちなみに転職者は年間{{< mark >}}331万人{{< /mark >}}（出典：厚生労働省「雇用動向調査」2023年実績）。転職はもう特別な行動ではないので、気負いすぎなくて大丈夫です。

{{< point >}}軸がないまま動くと、内定が出ても決められない。「なぜ・何を・どこまで」を最初に言語化する{{< /point >}}

## ②転職サイトとエージェントの違いを知る

「サイトとエージェント、どっちに登録すればいいの？」という疑問は、初心者がほぼ必ず通る道ですよね。

ざっくり言うと、転職サイトは自分で求人を探して応募する形式、転職エージェントは担当者（ようするに転職活動の伴走者）が求人紹介から面接調整まで無料で手伝ってくれる形式です。エージェントは{{< mark >}}非公開求人{{< /mark >}}（一般公開されていない求人で、全体の3〜5割）にアクセスできるのが大きな違いです。

{{< box color="blue" title="使い分けの目安" >}}
- まず市場を眺めたい → 転職サイトに登録して求人を見る
- 初めてで進め方が不安 → エージェントに1〜2社登録して相談する
- 本命の応募段階 → 両方を並行して使う（多くの転職者がこの形）
{{< /box >}}

詳しい違いは[転職エージェントと転職サイトの違いとは](/posts/2026-04-15-転職エージェントと転職サイトの違いとは-それぞれの特徴と上手な使い分け方/)で解説しています。エージェントを複数使うべきかどうかは[複数登録のメリット・デメリット](/posts/2026-04-14-転職エージェントは複数登録すべき-掛け持ちのメリット-デメリットと正しいやり方/)、初めて使うときの落とし穴は[初めて転職エージェントを使うときの注意点7つ](/posts/2026-06-16-初めて転職エージェントを使うときの注意点7つ-後悔しない準備と選び方/)にまとめてあります。

{{< point >}}サイトは自分で探す、エージェントは無料で伴走してくれる。初めてなら両方を組み合わせるのが定石{{< /point >}}

## ③職務経歴書・履歴書を先に整える

書類作成は、転職活動でいちばん腰が重い工程ですよね。「応募したい求人が見つかってから」作る人が多いのですが、これは順番が逆です。良い求人ほど掲載期間が短く、見つけてから書き始めると間に合わないか、慌てて質の低い書類を出すことになります。

先に8割の完成度で作っておき、応募先に合わせて残り2割を調整する。これが現実的な進め方です。

{{< box color="cyan" title="書類準備のチェックリスト" >}}
- 職務経歴を時系列で書き出す（担当業務・チーム規模・使ったスキル）
- 成果はできるだけ数字にする（件数・金額・改善率・期間）
- 自己PRは「企業が求める人物像」から逆算して書く
- 完成したらエージェントの添削を受けて客観的な目を入れる
{{< /box >}}

書き方の詳細は[職務経歴書の書き方完全ガイド](/posts/2026-05-21-職務経歴書の書き方完全ガイド-採用担当が見るポイントとテンプレート/)を参考にしてください。自分ひとりで仕上げるのが不安な方は、[エージェントの添削サポート](/posts/2026-05-09-職務経歴書はエージェントに添削してもらえる-効果は/)を使うと効率的です。私も添削を受けましたが、自分では気づけない「アピールの抜け」を指摘してもらえるのは大きかったです。

{{< point >}}書類は求人を見つける前に8割作っておく。数字での実績表現とエージェント添削で質を上げる{{< /point >}}

## ④在職中にスケジュールを組んで動く

「働きながらで本当に間に合うの？」という不安、よく分かります。結論から言うと、転職活動は**在職中に進めるのが原則**です。

データを見てみましょう。転職活動の期間は平均2〜3ヶ月（出典：doda「転職活動のスケジュール」）。想定期間として最も多いのは「3ヶ月以上半年未満」で28.1%です（出典：マイナビ「転職活動実態調査2025」）。さらに内定から入社までは平均49.0日かかります（出典：マイナビ「転職動向調査2025年版」）。つまり全体では{{< mark >}}3〜6ヶ月{{< /mark >}}を見込んでおくのが現実的です。

在職中は「時間がない」のが最大の壁ですが、日程調整や企業とのやり取りはエージェントが間に入ってくれます。私も在職中に活動しましたが、面談や面接の日程はほぼ調整してもらえたため、有給を数回使う程度で進められました。

働きながらの具体的な進め方は[在職中に転職活動はできる？](/posts/2026-04-19-在職中に転職活動はできる-転職エージェントを使って無理なく進める方法/)、期間の実態は[転職活動の平均期間](/posts/2026-04-21-転職活動の平均期間はどのくらい-エージェント利用者のリアルなデータと経験談/)で詳しく書いています。エージェントに登録した後の流れが不安な方は[初回面談で聞かれることと準備](/posts/2026-04-13-転職エージェントに登録したら何をする-初回面談で聞かれることと準備すべきこと/)を先に読んでおくと安心です。

{{< point >}}全体で3〜6ヶ月を見込み、在職中に進めるのが原則。日程調整はエージェントに任せれば負担は小さい{{< /point >}}

## ⑤応募・面接は並行で進めて比較する

応募の段階でやりがちな失敗が「本命1社だけ受けて結果を待つ」進め方です。1社ずつだと時間がかかるうえ、比較対象がないので条件の良し悪しも判断できません。2〜3社を並行して受けるのが基本です。

面接で必ず聞かれるのが転職理由です。①で言語化した軸を「前向きな動機」に変換して伝えます。「今の職場が嫌だから」ではなく「〇〇を実現したいから」という形です。答え方の具体例は[退職理由の伝え方｜面接でNGな言い方と好印象な例文集](/posts/2026-05-16-退職理由の伝え方-面接でNGな言い方と好印象な例文集/)にまとめています。

初めての内定は嬉しくて舞い上がってしまうものです。気持ちはよくわかります。ただ、承諾の前に労働条件を書面で確認してください。年収の内訳（基本給と残業代・賞与）、残業時間、入社日。口頭の説明と書面が食い違うことは珍しくありません。もし返事を急かされて困ったら、[内定承諾を急かされる理由と断り方](/posts/2026-05-08-転職エージェントに内定承諾を急かされる理由と断り方/)が参考になります。

複数の内定で迷ったときは、①の軸に戻って「譲れない条件」を満たす方を選びます。私は軸がなかったせいでここで立ち止まりました。同じ回り道をしないでほしい、というのがこの記事でいちばん伝えたいことです。

{{< box color="yellow" title="内定承諾前のチェック3点" >}}
- 労働条件通知書（雇用契約書）を書面でもらったか
- 年収の内訳・残業時間・入社日を確認したか
- ①で決めた「譲れない条件」を満たしているか
{{< /box >}}

{{< point >}}応募は2〜3社並行が基本。内定は書面の条件と「自分の軸」で判断すれば後悔しない{{< /point >}}

## ⑥円満退職の段取りで締めくくる

内定を承諾したら、最後のやるべきことが円満退職です。退職はなかなか言い出しにくいものですよね。ただ、ここを雑にすると、有給消化や退職日でもめて入社日に影響が出ることがあります。

退職の意思は、**まず直属の上司に、退職希望日の1〜2ヶ月前**を目安に伝えます。法律上は2週間前の申し出で退職できますが（民法627条）、引き継ぎを考えると余裕を持つ方が円満です。就業規則に「退職は◯ヶ月前までに申し出ること」と定められている場合も多いので、先に確認しておきましょう。

{{< box color="gray" title="円満退職の段取り" >}}
1. 就業規則の退職ルールを確認する
2. 直属の上司にアポを取り、口頭で退職意思を伝える
3. 退職日と有給消化のスケジュールを合意する
4. 引き継ぎ資料を作成し、後任へ引き継ぐ
5. 退職届の提出・備品返却・書類（離職票等）の受け取り
{{< /box >}}

強い引き止めにあっても、決断が固まっているなら感謝を伝えつつ意思は変えないことです。転職後の心構えも含めて、[転職して半年で辞めたくなったら](/posts/2026-06-09-転職して半年で辞めたくなったら-円満退職のタイミング/)も入社後にきっと役立ちます。

{{< point >}}退職意思は直属の上司へ1〜2ヶ月前に。就業規則の確認と引き継ぎの丁寧さが円満退職のカギ{{< /point >}}

## まとめ｜今日やることは「軸の言語化」から

{{< box color="cream" title="転職初心者がやるべきこと7つ・おさらい" >}}
1. **転職の軸を決める**（なぜ・何を・どこまで、を言語化）
2. 転職サイトとエージェントの違いを知る
3. 職務経歴書・履歴書を先に8割作る
4. 在職中にスケジュールを組む（全体3〜6ヶ月）
5. 応募・面接は2〜3社並行、内定は書面で確認
6. 円満退職の段取り（1〜2ヶ月前に上司へ）
