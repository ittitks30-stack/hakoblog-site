## ① M&Aコンサルタントとは？仕事内容と市場の爆発的成長

「そもそもM&Aコンサルって何をする仕事？」と思う方も多いですよね。まずここを整理しましょう。

### M&Aアドバイザリーの業務フロー（ソーシングからクロージングまで）

M&Aコンサルタントとは、企業の買収・売却・合併をサポートするプロフェッショナルです。ようするに「会社の売り買いの仲人」と考えるとわかりやすい。

業務の流れは大きく5段階に分かれます。

1. ソーシング：売り案件・買い案件の発掘（テレアポ・紹介・スカウトレターなど）
2. マッチング：売り手と買い手の相互紹介・意向確認
3. デューデリジェンス（DD）：財務・法務・ビジネスの詳細調査
4. バリュエーション：企業価値の算定（ようするに「この会社はいくらで売れるか」の計算）
5. クロージング：最終契約締結・成約報酬の受領

1〜2年目はソーシングが業務の大半を占めます。
「見込み客すら全くない状態で終日架電やDMに費やす日が続く」という現役コンサルの声もあります。

華やかなイメージとは裏腹に、初期は泥臭い営業活動が成功の土台になります。

{{< box color="gray" title="M&Aコンサルの3つの形態" >}}
- M&A仲介：売り手・買い手双方と契約し中立的に仲介。成功報酬型で高インセンティブ
- FAS（ファイナンシャルアドバイザリーサービス）：片方の代理人として純粋に助言。Big4系列が多い
- インハウスM&A：事業会社の経営企画・M&A専任部署。年収は抑えめだが安定性が高い
{{< /box >}}

この3つの形態の選択が、あなたのキャリアの方向性を大きく左右します。「自分がどのタイプに向いているか」を意識しながら読み進めてください。

### 市場規模の拡大と「今が転職のチャンス」な理由

レコフデータ運営「MARRオンライン」によると、2025年の国内M&A件数は5,115件で過去最多を記録し、取引金額は35.7兆円と歴代最高に達しました。

2022年の4,304件からわずか3年で20%近く増加しています。この需要の急拡大が、採用市場にも直接影響を与えています。

さらに見落とせないのが、中小企業の事業承継問題です。
社長年齢60歳以上の事業承継候補企業は約9万3,536社にのぼり、2035年頃にピーク（約9万5,234社）を迎えると予測されています。
向こう10年は、M&A需要が構造的に拡大し続ける計算です。

{{< box color="blue" title="M&A市場の現状（2025年最新データ）" >}}
- 年間M&A件数：5,115件（2025年・過去最多）
- 年間取引金額：35.7兆円（2025年・歴代最高）
- 事業承継候補企業：約9万3,536社（社長年齢60歳以上）
- 事業承継型M&A潜在市場：約13兆5,000億円
{{< /box >}}

出典：レコフデータ運営「MARRオンライン」2025年M&A回顧、日本経済新聞

{{< point >}}M&A市場は過去最高水準で拡大中。構造的な需要増が転職機会を広げている。{{< /point >}}

### 歩合・インセンティブ構造の実態

M&A仲介系では、成約1件で数百万〜数千万円の成功報酬が入ります。

大手食品メーカーで社内上位の営業成績を残してきた転職者が、M&A仲介会社へ転職後、入社2年目で3件を成約させ年収2,000万円を達成した事例があります（KOTORA JOURNAL）。

ただし、覚悟が必要な現実もあります。

- 1〜2年目は成約がなく離職する人も多い
- 月間残業時間71〜73時間が標準的（日本M&Aセンター実績）
- 繁忙期は月80〜100時間の残業も（OpenWork口コミ）
- 「ソーシングは終わりが見えない中で泳ぎ続けているよう」という声がある

転職後に新たな課題に直面した人は44.7%にのぼり、そのうち60.0%が「専門知識習得の難しさ」を痛感しています（転職後アンケート）。

それでも、70.6%が転職後に年収アップを実現し、23.5%は50%以上の増加を達成しています。

{{< box color="yellow" title="インセンティブ型報酬の注意点" >}}
- 固定給は一般サラリーマンより低いことが多い（年400〜600万円程度）
- 成功報酬が入るまで6〜18ヶ月かかるケースが多い
- 「高年収イメージだけで転職→固定給の低さに耐えられず離職」が最多の失敗パターン
- 住宅ローンや養育費がある30代は収入変動リスクを事前にシミュレーションすること
{{< /box >}}

{{< point >}}M&A仲介の年収2,000万円超は現実だが、そこに至るまでの低収入期間への備えが必須。{{< /point >}}

## ③ 30代未経験からM&Aコンサルに転職できるか？現実的な答え

### 採用されやすいバックグラウンドと「刺さるスキル」

「『30代の未経験は無理だ』と言われたんですが、本当ですか？」という声をよく聞きます。

正直に言います。**30代前半なら可能性は十分あります。**

採用においてコンサル業界内出身者は全体の約25%に過ぎず、約75%が異業種出身者です。
30代未経験でコンサル転職した人の割合は、全未経験転職者の約35.7%にのぼります（note調査）。
30代前半は、未経験転職全体の約3割を占める主要層でもあります。

ここで、他記事がほとんど触れていない「商社・メーカー出身者の強み・弱み」を解説します。

{{< box color="green" title="バックグラウンド別の強みと弱み" >}}
銀行・証券出身：
- 強み：財務分析・融資審査経験、コンプライアンス意識
- 弱み：営業スタイルが御用聞き型になりやすい

商社出身：
- 強み：業界知識の深さ、クロスボーダー交渉経験、事業全体の理解
- 弱み：財務モデリング（Excelモデル作成）の経験不足

メーカー営業出身：
- 強み：顧客課題の深掘り力、長期関係構築、数字へのコミットメント文化
- 弱み：M&A専門知識がゼロからのスタート

公認会計士・税理士：
- 強み：財務DD・バリュエーション即戦力。FAS系では圧倒的優位
- 弱み：営業経験の薄さ（仲介系ではハンデになることも）
{{< /box >}}

商社やメーカー営業出身者がM&A仲介を目指す場合、「業界知識×顧客との信頼構築力」が強力な武器になります。財務モデリングは入社後に学べますが、業界知識と営業力は後天的に身につきにくいからです。

### 未経験ゆえのハンデと乗り越え方

「30代後半での転職は書類選考通過率が相当低く、現実は甘くないと実感した」という声もあります。これは事実です。

30代前半（〜35歳）と後半（36歳以上）では、採用難易度に大きな壁があります。30代後半では未経験転職が1割以下に急減します。

ハンデを乗り越えるための具体的なアクションを挙げます。

1. 財務知識の補強：簿記2級取得（2〜3ヶ月で可能）
2. 業界研究の深化：自分の出身業界に特化した案件を訴求ポイントにする
3. 転職ストーリーの構築：「前職の経験×M&A」の接続を言語化する
4. エージェント選び：未経験30代専門のM&A特化エージェントを使う

4つ目が実は最重要です。エージェントの質が転職成否を大きく左右します（後述）。

### 実際に転職成功した30代の事例パターン

事例①：メーカー営業 → M&A仲介 → 入社2年目で年収2,000万円

大手食品メーカーで社内上位の営業成績を残してきた30代前半のAさん。M&A仲介会社に転職後、入社2年目で3件を成約させ年収2,000万円を達成しました（KOTORA JOURNAL掲載事例）。前職での「数字へのコミットメント」と「顧客との長期関係構築」が評価されました。

事例②：銀行員 → M&A仲介 → 1年で責任者候補に抜擢

30代前半で銀行勤務経験を持つBさんが、中堅M&A仲介会社に転職後、わずか1年で中堅事業会社のM&A責任者候補に抜擢。財務分析の基礎がある分、クライアントとの信頼構築が早かったのが決め手でした。

{{< box color="cyan" title="転職成功者の共通点3つ" >}}
- 前職の「専門業界×M&A」という訴求ストーリーが明確だった
- 自分を「即戦力」として売り込める側面を1つ以上持っていた
- M&A特化エージェント経由で非公開求人にアクセスできた
{{< /box >}}

{{< point >}}30代前半は未経験転職の「現実的なラストチャンスゾーン」。早めの行動が成否を分ける。{{< /point >}}

外資系IBD（インベストメント・バンキング部門、ようするに外資投資銀行のM&A担当）のMDは年収4,800万円以上にのぼります。

30代未経験で入社した場合、通常はアナリストまたはアソシエイトポジションからスタートします。3〜5年かけてマネージャーに昇格すれば、1,200万〜2,000万円のレンジに到達できます。年齢的には35〜40歳前後。現実的なゴールとして設定できる水準です。

### M&Aコンサル後のExit先（PE・事業会社役員・独立）

「M&Aコンサルに入った後どこに行けるのか」は、転職を決断するうえで非常に重要な視点です。競合記事がほぼ触れていない独自の切り口も含めて解説します。

主なExit先は3つあります。

- PEファンド（プライベートエクイティ）：M&Aコンサル経験者の最もプレステージが高い転職先。年収は2,000万〜5,000万円超。FAS系出身者は案件分析力が評価され、M&A仲介出身者は案件ソーシング能力が評価される
- 事業会社役員・CFO：M&Aの実行経験を持つ人材は事業会社の経営層として引く手あまた。大手上場企業のCFOやM&A担当役員を目指せる
- 独立・M&Aアドバイザー：実績を積んだ後に個人でアドバイザリーフィーを得るルート。仲介系出身者に多い

注目すべきは「M&A仲介系」と「FAS系」でExit先の質が異なる点です。
PEファンドへの転職難易度は「FAS出身 > M&A仲介出身」。
一方、事業会社CFO・役員への転職では「M&A仲介出身の方が実行経験を評価される」傾向があります。

{{< box color="gray" title="Exit先の選び方" >}}
- 将来PEファンドへ行きたい → FAS（Big4系列）を狙う
- 事業会社の経営に関わりたい → M&A仲介か事業会社M&A部門が近道
- とにかく年収を最大化したい → M&A仲介 → PEファンドの「最強ルート」
{{< /box >}}

### MBAは必要か？取得のコスパ検証

「M&AコンサルはMBAが必要ですか？」という質問をよく受けます。結論から言うと、取るかどうかはファームの種類と志望ポジションによります。

キャリアインキュベーション株式会社の調査（2020〜2021年実施）によると、MBA取得後の初回転職で年収500万円以上アップした人が37.9%、300万円以上アップが57.1%という結果が出ています。
40歳時点での年収1,000万円超は80.7%と、同年代平均の2倍以上です（出典：PR TIMES）。

ただし、MBA不要なケースも明確にあります。

| ルート | MBA必要性 |
|---|---|
| M&A仲介（日系大手）| 不要。営業実績・コミュ力重視 |
| FAS系（Big4）| 会計士資格の方がMBAより評価される |
| 外資系IBD | 有効。ブランド校MBAが採用で有利 |
| インハウスM&A | 不要。実務経験重視 |

**M&A仲介を目指すなら、MBAよりも簿記2級と転職ストーリーの磨き込みを先に行う**のが合理的な判断です。MBAに2〜3年かけるより先に転職活動を始めた方が年齢的に有利なケースが多い。もちろん絶対ではありませんが。

{{< point >}}MBAの効果は本物だが、志望先によっては不要。「まず転職、MBAは後から」という順序も十分あり得る。{{< /point >}}
