「外資系に転職したいけど、英語が全然できなくて…」

そのひと言で、外資系転職を諦めていませんか。実際には、TOEICが600点台でも、英語面接が不得手でも、外資系企業に転職している人は大勢います。

エージェント利用経験のある運営者ittiが、実体験をもとに解説します。この記事では次の3点を中心にまとめました。

- 職種・ポジション別に必要なTOEICスコアの目安
- 英語が苦手でも入社できるケースと条件
- 外資系転職に強いエージェントの選び方と使い方

数字と具体例を中心に、外資系転職の現実を正直に書きました。

---

## 外資系転職に本当に必要な英語力とは

「外資系＝英語ペラペラ必須」というイメージは、半分正解で半分誤解です。そう思い込んでいる方は多いですよね。結論から言うと、**職種とポジションによって求められる英語レベルは大きく異なります**。

### 職種・ポジション別のTOEIC目安スコア

JACリクルートメントおよびエンワールドの情報をもとに整理しました（出典：JACリクルートメント「外資系企業への転職とTOEIC」、エンワールド「外資系企業への転職で要求される英語力」）。

| 職種・業界 | TOEICスコア目安 |
|---|---|
| 外資系全般（書類選考通過ライン） | 600〜700点以上 |
| 外資系メーカー・技術職 | 700〜730点以上 |
| 外資系ITエンジニア職 | 730点程度 |
| 外資系営業 | 800点程度 |
| 外資系コンサルティング | 800〜900点以上 |
| 外資系金融 | 800〜900点以上 |
| 海外拠点勤務を伴うポジション | 850点程度 |

{{< box color="blue" title="ポジション別の考え方" >}}
- 一般社員レベル：500点台以上でも可
- マネージャー・ディレクタークラス：700点以上が多い
- 「400点でも向上心があれば評価対象」という人事担当者の声もある

（出典：IIBC公式ブログ「外資系企業人事担当者に聞いたTOEICとリアルな採用事情」）
{{< /box >}}

ここで重要なのは「スコアが低くても合否を左右するのは別の要因」という点です。

### 英語力よりも重視されるスキル・経験

IIBC（国際ビジネスコミュニケーション協会）が外資系企業の人事担当者に行った調査では、「TOEICスコアはあくまで参考指標」という認識が多数でした（出典：IIBC公式ブログ）。実際に採用現場で重視されるのは以下の3点です。

- 専門スキルや職種固有の実績（売上数字・プロジェクト規模など）
- 「これから伸びる」という向上意欲と学習姿勢
- 担当業務で英語をどの程度使うかという業務要件との一致

私自身、製造業の機械加工出身でJACリクルートメントを利用した際、コンサルタントから言われたのは「英語よりも現場でどれだけ価値を出せるかを先に見られます」という一言でした。英語は入社後に伸ばせるが、専門性は短期では補えない、という判断です。

外資系転職において、英語力は「必要条件」ではあっても「決定要因」になることは少ないのが現実です。

{{< point >}}職種・ポジションによってTOEIC目安は500〜900点と幅広く、英語より専門スキルと実績が採用の軸になる{{< /point >}}

## 英語が苦手でも外資系に入れる？ケース別の現実

「英語できないのに外資系に転職できた人って本当にいるの？」と思いますよね。います。条件次第では英語力をほぼ問われないポジションも存在します。

### 英語不要ポジションは本当にあるのか

英語不要・英語力不問の外資系求人は実在します。代表的な職種は以下の通りです（出典：マイナビ転職グローバル「英語力不問」求人掲載情報）。

- アパレル系外資の国内販売・接客職（顧客は主に日本語話者）
- 製薬メーカー外資のMR（国内医療機関が対象のため日本語中心）
- 製造部門・物流部門のバックオフィス（社内の英語接点が少ない）

ただし注意点があります。英語不要で入社できても、昇進や管理職への道では英語力が求められることが多い。「今は英語なしでいいが、3〜5年後には必要になる」と覚悟しておく必要があります。

{{< box color="yellow" title="英語不要ポジションの落とし穴" >}}
- 昇格条件にTOEICスコアが設定されているケースがある
- グローバル会議への参加機会が増えると英語対応が避けられなくなる
- 英語学習を先送りにすると、後から追いつくのが一層大変になる
{{< /box >}}

### 入社後に英語力を伸ばすパターン

実際に英語力が不十分なまま外資系に入社し、その後伸ばした人たちには共通点があります。転職成功者の体験談（出典：note「外資系転職体験記」）をもとに整理すると、以下の3つのアプローチが有効でした。

1. **日英両言語で職務経歴書を作成**し、書くことで英語の語彙を業務に紐付ける
2. 面接・会議では「日本語で内容を整理してからシンプルな英語に変換」する思考法を習慣化する
3. 英語の弱みを隠さず「これから伸ばします」という意欲を積極的に示す

採用担当者が見ているのは「今できるかどうか」だけでなく「これから伸びる人かどうか」です。入社後の成長可能性が見える人は英語が弱くても通過するケースがあります。

{{< point >}}英語不要ポジションは存在するが、昇進では必要になることが多い。入社後に伸ばす前提で準備しておくことが重要{{< /point >}}

### エージェントを選ぶ3つの軸

外資系転職のエージェント選びは、以下の3軸で考えると迷いにくくなります。

- **年収帯が合っているか**：JACは年収600万円以上が中心。年収400〜500万円台だと紹介できる求人が限られるという声もある
- **英語サポートが必要か**：英文CV添削・英語面接対策まで必要なら、JACかマイケルページが有力候補
- **外資系一本に絞るか、日系も含めるか**：外資系一本ならJAC・ロバートウォルターズ・マイケルページ。日系も視野に入れるならリクルートエージェントを並行利用

{{< box color="cyan" title="2段活用戦略" >}}
まずビズリーチ（スカウト型）に登録して市場価値・スカウト数を確認する。その後、JACに本格登録する——この順番で動くと、自分の現在地を把握した上でエージェントと話せるため、交渉力が上がり外資系転職の成功率が高くなる。

（出典：ビズリーチ公式、JACリクルートメント公式）
{{< /box >}}

### エージェント活用の注意点

ネガティブな声も正直に伝えます。

「担当者によって対応の質に大きな差がある。合わない担当になると転職活動が停滞する」という声は多い。もし初回面談で「紹介できる求人がない」と言われた場合、別のエージェントにすぐ切り替えることをためらわないでください。1社で止まらず、2〜3社を並行登録するのが基本です。

{{< point >}}外資系転職はJACを起点に2〜3社並行登録が定石。2段活用（ビズリーチで市場価値確認→JAC本登録）で成功率が上がる{{< /point >}}

## 外資系転職の選考プロセスと対策

「英語面接は何を聞かれるの？」という不安、よく分かります。外資系の選考は日系企業とは異なる点が複数あります。事前に把握しておくと、準備の質が上がります。

### 英語面接・ケース面接の傾向

外資系企業の面接は、大きく2種類に分かれます。

1. **英語面接**：英語で自己紹介・職歴説明・志望動機を問われる。コンサル・金融は英語100%のケースも
2. **ケース面接**：コンサルティング会社で頻出。「市場規模を推定せよ」「この企業の課題を解決せよ」といった問題を議論する形式

英語面接で重要なのは「流暢さ」より「論理の明快さ」です。「日本語で整理した内容をシンプルな英語に落とし込む」思考法が実践的です（出典：note「外資系転職体験記」）。

{{< box color="green" title="英語面接で使える準備法" >}}
- 職歴・実績を英語で60秒にまとめる練習（STAR法：状況→課題→行動→結果）
- 「現在の課題」「転職理由」を英語で3文以内に整理する
- 英文CVと話す内容を一致させておく（矛盾が信頼感を下げる）
{{< /box >}}

### 実体験から見た選考で見られるポイント

私がJACを通じて外資系含む複数社の面接を経験した際、採用担当者が繰り返し掘り下げたのは「過去の具体的な成果の数字」でした。「何人のチームで、どの程度の予算を動かし、結果として何%改善したか」という定量的な説明を求められました。

外資系では「自分の成果を自分の言葉で語れること」が基本姿勢です。日系企業に多い「チームで頑張りました」という答えは、外資系では評価されにくいとJACのコンサルタントから事前に教わりました。

英語面接では、このSTAR法のフレームを英語で繰り返せるよう練習しておくと、内容の迷いがなくなり言葉が出やすくなります。

{{< point >}}英語面接は「流暢さ」より「論理の明快さ」が優先。STAR法で自分の実績を数字込みで準備するのが最優先{{< /point >}}

## 外資系への転職を決める前に確認すべきこと

転職後に「こんなはずじゃなかった」とならないために、外資系の働き方の現実も把握しておいてください。入社前に知っておくかどうかで、転職後の満足度はかなり変わりますよね。

### 文化・働き方の違い（Up or Out / 成果主義）

外資系企業には独自の文化があります。代表的な2点を正直に書きます。

**Up or Out（昇進か退場か）**とは、一定期間内に昇進できない場合は退職を促されるという仕組みです。ようするに「結果を出し続けなければポジションを失う」という文化です。コンサルティングファームや外資系金融で特に顕著です。

**成果主義**は、年功序列ではなく実績・成果に対して報酬が決まる仕組みです。同期でも年収に大きな差が生まれます。

{{< box color="yellow" title="外資系転職前に確認しておくべき4点" >}}
- 業績評価の頻度と基準（半期・年次・四半期など）
- 解雇・リストラリスクの高さ（外資系の組織再編は日系より速い）
- 有給消化・残業文化（外資系でも部署によって大きく異なる）
- 日本法人の規模と本社からの裁量（100人未満の小規模法人は異動・昇進の選択肢が少ない）
{{< /box >}}

### 年収・福利厚生の現実

外資系の年収は平均的に高い。JACリクルートメントのデータによると、外資系全体の平均年収は<strong>915.8万円</strong>（日本全体平均478万円の約2倍）です（出典：JACリクルートメント「外資系企業の年収」）。年代別では以下の通りです。

| 年代 | 外資系平均年収 | 日系との差 |
|---|---|---|
| 20代 | 約600万円 | +40万円 |
| 30代 | 約700万円 | +30万円 |
| 40代 | 約900万円 | +60万円 |
| 50代 | 約1,050万円 | +130万円 |

もちろん人によりますが、ハイクラス職種ではコンサル・投資銀行で30歳前後に年収1,000万円超えも珍しくありません。

一方で、退職金制度が日系より薄い、福利厚生が最低限というケースも多い。年収の「額面」だけで比較すると入社後に後悔することがあるため、**総報酬（ベース＋ボーナス＋退職金制度）で比較する**ことが重要です。
